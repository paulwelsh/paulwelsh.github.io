eWeb_CheckForUpdate

Checks for an available update for a given SKU.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at Installing eWebControl365.dll (the recommended location). It may be already placed here by another eSellerate-enabled application.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

Integer eWeb_CheckForUpdate ( esellerID, SkuRefNum, previewID, layoutCertificate, trackingID, affiliateID, couponID, activationID, extraData, resultData )

esellerID – eSeller ID code

SkuRefNum – SKU RefNum Code

previewID – Preview Certificate string (optional)

layoutCertificate – Layout Test Certificate string (optional)

trackingID – tracking ID code (optional)

affiliateID – affiliate ID code (optional)

couponID – coupon ID code (optional)

activationID – Product Activation Set ID string (optional)

extraData – special serial number data string (optional)

resultData – reference to eWebResultData object


See eSellerate Return Values for details on return values.

Parameter

Description

esellerID

String containing the Embedded Web Store ID, as found on the Sales Manager Embedded Web Store eSeller Summary page. Required.

SkuRefNum

String containing the SKU RefNum, as found on the Sales Manager Embedded Web Store SKU Attributes page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Embedded Web Store eSeller Summary page. Required to purchase in Preview mode. Pass "" in Active mode. If the Preview Certificate passed here does not match the Preview Certificate on the Sales Manager Embedded Web Store eSeller Summary page, the purchase will be Active. This means that you can clear the Preview Certificate in the Sales Manager to activate the eSeller without having to recompile your application.

layoutCertificate

String containing the Layout Test Certificate, as found on the Settings tab of the Custom Layout Detail page. Allows testing of unpublished Custom Layouts. May be "". If the Layout Test Certificate passed here does not match the one on the Settings tab of the Custom Layout Detail page, the eSeller will use the published version of the Custom Layout. This means that you can clear the Layout Test Certificate in the Sales Manager to use the published layout without having to recompile your application.

trackingID

String containing special information that marks the function call so that it can be tracked in reports. May be "". See "tracking IDs" in the eSellerate User's GuidePublishers Help index.

affiliateID

String containing an affiliate ID that marks the function call so that it can be tracked in reports. May be "". For information on the eSellerate Affiliates program, see "eSellerate Affiliates: joining" in the eSellerate User's GuidePublishers Help index.

couponID

String containing a coupon ID that enables coupon discounts on qualified purchases. May be "". See "coupons: creating" in the eSellerate User's GuidePublishers Help index.

activationID

String containing an activation ID for use in eSellerate Product Activation. May be "". Automatic activation occurs if the Embedded Web Store Engine completes the action and returns a unique serial number for use with the activationID (barring any activation errors). See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

extraData

String containing special additional information needed for serial number generation. Required for serial number type option Application Supplied and optional for Standard, Standard Enhanced and Armadillo. For all other serial number type options, pass NULL. See "serial numbers: Serial Number Type options" in the eSellerate User's GuidePublishers Help index.

resultData

eWebResultData object passed by reference. Required.
