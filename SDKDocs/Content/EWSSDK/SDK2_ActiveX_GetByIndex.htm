eWeb_GetOrderItemsByIndex

Extracts information about a single order item from resultData. See also eWeb_GetResultDataValue and eWeb_IndexGetResultDataValue.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at Installing eWebControl365.dll (the recommended location). It may be already placed here by another eSellerate-enabled application.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

eWeb_GetOrderItemsByIndex ( Index, SkuRefNum, RedirectSkuRefNum, Quantity, RegistrationName, SerialNumber, PromptedValue, ActivationID, ActivationKey, DownloadURL )

Index – zero-based index of the order item for which you are searching

SkuRefNum – original SKU purchased (optional)

RedirectSkuRefNum – SKU RefNum that was actually purchased in case of a SKU redirect (optional)

Quantity – quantity of the item being purchased (optional)

RegistrationName – name on which the serial number was based (optional)

SerialNumber – serial number associated with this purchase (optional)

PromptedValue – prompted value entered by the user if one was requested (optional)

ActivationID – activation ID for the product (optional)

ActivationKey – activation key for the product (optional)

DownloadURL – eSellerate URL from which this product may be downloaded (optional)


See eWebResultData Attributes for information on attributes that can be retrieved directly from the eWebResultData class without having to call eWeb_GetOrderItemsByIndex.

For a listing of all eSellerate variables that Embedded Web Store eSellers can prefill or pull from result data, use the following URL: http://publishers.esellerate.net/embeddedvar.aspx.

See eSellerate Return Values for details on return values.

Parameter

Description

Index

Zero-based index of the order item for which you are searching. Required.

SkuRefNum

SKU RefNum for the original SKU purchased. May be "" if this information is unneeded.

RedirectSkuRefNum

SKU RefNum for the redirect SKU. May be "" if this information is unneeded.

Quantity

Quantity of the item being purchased. May be "" if this information is unneeded.

RegistrationName

Name on which the serial number was based. May be "" if this information is unneeded.

SerialNumber

Serial number associated with the order item. May be "" if this information is unneeded.

PromptedValue

Prompted value entered by the user for use in serial number generation. May be "" if this information is unneeded.

ActivationID

Activation ID for the order item. May be "" if this information is unneeded.

ActivationKey

Activation key for the order item. May be "" if this information is unneeded.

DownloadURL

eSellerate URL from which this order item may be downloaded. May be "" if this information is unneeded.
