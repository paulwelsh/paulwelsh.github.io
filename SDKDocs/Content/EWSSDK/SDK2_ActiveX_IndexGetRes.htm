eWeb_IndexGetResultDataValue

Returns the data for the supplied record/field combination.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at Installing eWebControl365.dll (the recommended location). It may be already placed here by another eSellerate-enabled application.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

Integer eWeb_IndexGetResultDataValue ( Record, Field, Index, Value )

Record – name of the record that contains the value whose length is being searched

Field – name of the field within the record for which you wish to search

Index – zero-based index of the name instance

Value – buffer to receive the value upon success


See eWebResultData Attributes for information on attributes that can be retrieved directly from the eWebResultData class without having to call eWeb_IndexGetResultDataValue.

For a listing of all eSellerate variables that Embedded Web Store eSellers can prefill or pull from result data, use the following URL: http://publishers.esellerate.net/embeddedvar.aspx.

See eSellerate Return Values for details on return values.

Parameter

Description

Record

Name of the record containing the variable being searched. Required. An example of a valid record name follows:

Complete variable name: _CartItem.Quantity
Record name: _CartItem

Field

Name of the field within the record for which you wish to search. Required. An example of a valid field name follows:

Complete variable name: _CartItem.Quantity
Field name: Quantity

Index

Zero-based index of the record/field instance. Required.

Value

Buffer to receive the value upon success. Required.
