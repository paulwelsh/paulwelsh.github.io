eWeb_IsSystemCompatible

The eWeb_IsSystemCompatible function determines whether the operating system will support the Embedded Web Store Engine. In the event that the function determines the system is incompatible, we recommend launching a standard eSellerate Web Store to allow your customers to complete their purchases.

Required Files

  • eWebControl365.dll
    Included in the development project.

Definition

Boolean eWeb_IsSystemCompatible ()


See eSellerate Return Values for details on return values.

Example

Dim eSellerate as eWebSDKLib.eWebSDK

esellerate = CreateObject("eWebSDK.365")

 

if (esellerate.IsSystemCompatible = true) then

' All embedded eSellerate calls are available

End If
