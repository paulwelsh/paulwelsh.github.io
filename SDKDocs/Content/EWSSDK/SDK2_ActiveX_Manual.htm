eWeb_ManualActivateSerialNumber

Launches the Embedded Web Store Engine to allow clients to manually activate a serial number. Typically this API is used when the current machine cannot get a network connection and a serial number needs to be activated.

Also see "Product Activation: manually activating serial numbers" in the eSellerate User's GuidePublishers Help index.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at See "Installing eWebControl365.dll" (the recommended location). It may be already placed here by another eSellerate-enabled application.
  • eWebClient.dll
    Located in the eSellerate directory noted at See "Default Locations for eWebClient.dll". It may be already be placed here by another eSellerate-enabled application.

Definition

Integer eWeb_ManualActivateSerialNumber ( publisherID, activationID, serialNumber, titlePreface, otherOptions )

publisherID – Publisher ID string

activationID – Product Activation Set ID string

serialNumber – serial number string

titlePreface – custom text, to preface activation wizard titles (optional)

otherOptions – custom text, to include phone number, e-mail or other contact (optional)


See eSellerate Return Values for details on return values.

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

titlePreface

String containing a custom preface to the titles of all Manual Product Activation Wizard pages. Causes the titles to be "<Preface>: Product Activation." May be "", in which case the titles will simply be "Product Activation."

otherOptions

String containing custom information to help users without web access activate their software. (This might be contact information for the publisher, who would visit http://activate.esellerate.net to obtain an Activation Key for the user.) The wizard will display up to six lines for this string. May be "", in which case the option to activate without web access will not be available to users.
Parameters for eWeb_ManualActivateSerialNumber Function

Note: Unless there is a specific need for your application source code to directly invoke the Manual Product Activation Wizard, it is more efficient to use the eWeb_ActivateSerialNumber function as the primary activation method instead of eWeb_ManualActivateSerialNumber.

For example, you can configure your source code to use eWeb_ActivateSerial-Number to handle normal (automatic) activations. If the results returned from eWeb_ActivateSerialNumber indicate a network related failure, your source code can then call eWeb_ManualActivateSerialNumber.

Example

Dim esellerate As eWebSDKLib.eWebSDK

Dim activationResult As eWebSDKLib.eWeb_ReturnValues

Dim serialNumber as string = "xxxxx" ' fill in your serial number here

 

esellerate = CreateObject("eWebSDK.365")

activationResult = esellerate.eWeb_ManualActivateSerialNumber("PUB775527079", "ACT123456789:QTNCLL-AVKJ-A73C62-EXPN-ECEBTK-89U5R0-VN2D-FCDW7J-UVTA-B1PHCA", serialNumber, "GeckoSoft", "")
