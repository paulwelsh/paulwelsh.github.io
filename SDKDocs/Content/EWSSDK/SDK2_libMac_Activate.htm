eWeb_ActivateSerialNumber

Activates a serial number for use on the current host machine using eSellerate Product Activation.

Required Files

  • EWSLib.h
    Included in the development project.
  • libEWS.a (standard) or
    libEWS-GC.a (garbage collecting)
    Included in the development project.
  • EWSMac.framework (standard) or
    EWSMac-GC.framework (garbage collecting)
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.

Definition

OSStatus eWeb_ActivateSerialNumber ( publisherID, activationID, serialNumber, askToConnect )

publisherID – Publisher ID string

activationID – Product Activation Set ID string

serialNumber – serial number string

askToConnect – display the Manual Product Activation Wizad if no connection is available


See eSellerate Return Values for details on return values.

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

askToConnect

Logical value that controls whether the Manual Product Activation Wizard is displayed if no active connection to the Internet is found. Pass a TRUE value to enable display of the wizard; otherwise pass FALSE.
Parameters for eWeb_ActivateSerialNumber Function

Note: All strings passed into the engine are UTF-8 encoded cStrings.

Caution: Serial numbers used in Product Activation are case sensitive. This restriction is typically not an issue for SDK functions that handle serial numbers without user input, but it can be an issue if you use the eWeb_ActivateSerialNumber function to allow customers to activate software purchased through a web store. To help prevent E_ACTIVATESN_UNKNOWN_SN errors in this instance, you can configure your source code to correct any case discrepancies before passing a serialNumber parameter for the eWeb_ActivateSerialNumber function. Also, in the prompt for your customers to enter the serial number, you can include language such as: "Enter the serial number exactly as shown on your receipt, including case (uppercase or lowercase)."

A related precaution that you can take to help prevent errors is to remove any leading and trailing spaces from the user input before passing the serialNumber parameter.

Example

#include "EWSLib.h"

char *serialNumber;

OSStatus resultCode;

/* TO DO: obtain serialNumber */

resultCode = eWeb_ActivateSerialNumber("PUB775527079", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU", serialNumber, false);

/* TO DO: handle error, or react to success */
