eWeb_ConfirmSerialNumber

Confirms that a serial number is valid within the eSellerate system. A serial number is invalid if it is unknown, blacklisted or associated with an invalid order.

Required Files

  • EWSEmbedded.framework (standard) or
    EWSEmbedded-GC.framework (garbage collecting)
    Included in the development project.

Definition

OSStatus eWeb_ConfirmSerialNumber ( publisherID, serialNumber )

publisherID – Publisher ID string

serialNumber – serial number string


See eSellerate Return Values for details on return values.

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

serialNumber

String containing the serial number. Required.
Parameters for eWeb_ConfirmSerialNumber Function

Note: All strings passed into the engine are UTF-8 encoded cStrings.

Note: Because some types of serial number sets permit duplicate serial numbers, eWeb_ConfirmSerialNumber does not check to see if a serial number is associated with more than one order.
