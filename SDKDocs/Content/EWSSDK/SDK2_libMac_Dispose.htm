eWeb_DisposePrefillData

Disposes of memory allocated by eWeb_NewPrefillData and used by any routines that consume prefill data. This function must be called on the prefill data for it to be properly disposed.

Required Files

  • EWSLib.h
    Included in the development project.
  • libEWS.a (standard) or
    libEWS-GC.a (garbage collecting)
    Included in the development project.
  • EWSMac.framework (standard) or
    EWSMac-GC.framework (garbage collecting)
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.

Definition

eWeb_DisposePrefillData ( prefillData )

prefillData – prefill data buffer returned by eWeb_NewPrefillData


See eSellerate Return Values for details on return values.

Example

#include "EWSLib.h"

char *resultData = NULL;

char *prefillData = eWeb_NewPrefillData();

/* TO DO: use eWeb_AddPrefillData() to prefill information for the purchase */

/* TO DO: use prefillData in a function that uses it */

/* TO DO: handle error or utilize result data */

eWeb_DisposePrefillData(prefillData);

eWeb_DisposeResultData(resultData);
