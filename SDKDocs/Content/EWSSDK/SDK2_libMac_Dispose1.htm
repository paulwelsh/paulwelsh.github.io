eWeb_DisposeResultData

Disposes of memory allocated by API that return result data. This function must be called on the result data for it to be properly disposed.

Required Files

  • EWSLib.h
    Included in the development project.
  • libEWS.a (standard) or
    libEWS-GC.a (garbage collecting)
    Included in the development project.

Definition

eWeb_DisposeResultData ( resultData )

resultData – results returned from the Purchase or CheckForUpdate family of functions


See eSellerate Return Values for details on return values.

Parameter

Description

resultData

Address of a declared variable of type char *. This should have already been filled by eWeb_Purchase or eWeb_CheckForUpdate.

#include "EWSLib.h"

char *resultData = NULL
