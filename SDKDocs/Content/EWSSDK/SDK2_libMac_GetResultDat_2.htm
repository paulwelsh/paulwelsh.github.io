eWeb_GetResultDataValue

Retrieves the value of a given result data variable. In order to allocate enough memory for the result data variable's value, call eWeb_GetResultDataValueLength.

Tip: This API can retrieve the value of indexed and non-indexed eSellerate variables. Indexed variables are designated as "Loop Only" on the variables page at http://publishers.esellerate.net/embeddedvar.aspx .

Required Files

  • EWSEmbedded.framework (standard) or
    EWSEmbedded-GC.framework (garbage collecting)
    Included in the development project.

Definition

OSStatus eWeb_GetResultDataValue ( resultData, Name, Value, ValueLen )

resultData – results obtained from the Purchase or CheckForUpdate family of functions

Name – name of the result data variable for which to search

Value – memory region to hold the result data variable's value

ValueLen – length of the memory region to hold the result data variable's value


For a listing of all eSellerate variables that Embedded Web Store eSellers can pull from result data, use the following URL: http://publishers.esellerate.net/embeddedvar.aspx.

See eSellerate Return Values for details on return values.

Parameter

Description

resultData

Address of a declared variable of type char *. This should have already been filled by eWeb_Purchase or eWeb_CheckForUpdate.

Name

The name of the result data variable for which to search. Required. Must include a zero-based index if the variable is designated as "Loop Only" on the variables page at http://publishers.esellerate.net/embeddedvar.aspx. For example, use "_OrderItem0.SerialNumber" to search for the serial number issued for the first order item.

Value

Memory region to hold the result data variable's value. Required.

ValueLen

Length of the memory region to hold the result data variable's value. Required.
Parameters for eWeb_GetResultDataValue Function

Note: All strings passed into the engine are UTF-8 encoded cStrings.

Example

#import<EWSEmbedded/EWSEmbedded.h>

char *resultData = NULL;

char *serialNumber = "";

/* TO DO: obtain resultData by calling an Embedded Web Store invoking function (Purchase, CheckForUpdate) */

OSStatus resultCode = eWeb_GetResultDataValue(resultData, "_OrderItem0.SerialNumber", serialNumber, 64);

/* utilize the value you retrieved */

eWeb_DisposeResultData(resultData);
