eWeb_Today

Returns the current date, expressed as the number of days since Jan. 1, 2000.

Tip: eWeb_Today is for finding the current date when you're using the purchase library (EWSEmbedded.framework/EWSEmbedded-GC.framework). To find the current date with the validation library for Standard or Standard Enhanced serial numbers (libValidateUniversal.a), you can use eSellerate_Today. See eSellerate_Today.

Required Files

  • EWSEmbedded.framework (standard) or
    EWSEmbedded-GC.framework (garbage collecting)
    Included in the development project.

Definition

unsigned long eWeb_Today ()
