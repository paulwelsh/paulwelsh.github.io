eWeb Libraries for Windows

This section details the eSellerate Embedded Web Store SDK provided by the static library option for eSeller integration into Windows applications developed using Visual C++. It divides the functions into three categories: Standard, Extended and Results.

Standard

Standard functions enable everything from installing the Embedded Web Store Engine to software purchasing and activation. Some of these functions return result data, which can then be accessed through Results functions. The Standard functions are:

  • eWeb_IsSystemCompatible
  • eWeb_InstallEngineFromResource
  • eWeb_InstallEngineFromPath
  • eWeb_ApplicationLocation
  • eWeb_Purchase
  • eWeb_CheckForUpdate
  • eWeb_SilentCheckForUpdate
  • eWeb_WebStoreURL
  • eWeb_ActivateSerialNumber
  • eWeb_ManualActivateSerialNumber
  • eWeb_ValidateActivation
  • eWeb_DateActivation
  • eWeb_DeactivateSerialNumber
  • eWeb_ConfirmSerialNumber
  • eWeb_Today
  • eSellerate_ValidateSerialNumber
  • eSellerate_StandardizeSerialNumber
  • eSellerate_Today

Extended

Extended functions enable prefilling data to simplify the purchase process for customers. Some examples of prefill capabilities include prefilling the customer's billing information, adding one or more SKUs to the shopping cart, and activating all SKUs purchased. The Extended functions are:

  • eWeb_NewPrefillData
  • eWeb_AddPrefillData
  • eWeb_PurchaseEx
  • eWeb_CheckForUpdateEx
  • eWeb_DisposePrefillData

Results

Results functions enable access to result data returned through Standard and Extended functions that invoke the Embedded Web Store Engine. The Results functions are:

  • eWeb_GetOrderItemByIndex
  • eWeb_GetResultDataValueLength
  • eWeb_GetResultDataValue
  • eWeb_IndexGetResultDataValueLength
  • eWeb_IndexGetResultDataValue
  • eWeb_DisposeResultData
Notes
  • The system requirements for this option are Windows XP or later, or Windows 2000 or later, and Internet Explorer 5.5 or later.
  • For a summary of previous and current function calls applicable to eWeb libraries for Windows, see New Functions Summary.
  • The static library option of the Embedded Web Store SDK for Windows is compatible with 64-bit Windows editions such as Windows XP Professional x64 Edition. This is not a 64-bit solution, but you can integrate it with your 32-bit applications to allow Embedded Web Store functionality on 64-bit Windows.
  • Beginning with the version 3.6.5 SDK, the default install location for the eSellerate Embedded Web Store Engine (eWebClient.dll) is different for Vista than other Windows versions. The eWeb_InstallEngineFrom-Resource and eWeb_InstallEngineFromPath functions have been updated to install the engine in the proper locations. These locations are the ProgramData\eSellerate directory on Vista, and the Common Files\eSellerate directory on pre-Vista Windows.
  • Beginning with the version 3.6.5 SDK, it is no longer necessary to register the eSellerate Embedded Web Store Engine (eWebClient.dll). The eWeb_Install-EngineFromResource and eWeb_InstallEngineFromPath functions have been updated for this change.
