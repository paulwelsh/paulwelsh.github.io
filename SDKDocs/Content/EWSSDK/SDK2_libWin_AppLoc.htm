eWeb_ApplicationLocation

Returns the path to the application module calling the Embedded Web Store Engine.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

HRESULT eWeb_ApplicationLocation ( path, bufferLength )

path – file location to be filled as a result

bufferLength – size (in bytes) of space allocated for path


See eSellerate Return Values for details on return values.

Parameter

Description

path

File location to be filled as a result. Required.

bufferLength

Size (in bytes) of space allocated for path. Required.
Parameters for eWeb_ApplicationLocation Function

Example

#include "eWebLibrary.h"

char[500] path = "";

HRESULT resultCode = eWeb_ApplicationLocation(path, 500);
