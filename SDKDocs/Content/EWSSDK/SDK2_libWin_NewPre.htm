eWeb_NewPrefillData

Allocates a memory region for use by prefill data. Before passing prefill data to an Embedded Web Store API you must allocate memory for it using eWeb_NewPrefillData.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Example

#include "eWebLibrary.h"

BYTE *prefillData = eWeb_NewPrefillData();
