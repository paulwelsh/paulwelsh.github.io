eSellerate_Today

Returns the current date, expressed as the number of days since Jan. 1, 2000.

Tip: eSellerate_Today is for finding the current date when you're using the validation library (validateLibrary.lib) for Standard or Standard Enhanced serial numbers. To find the current date with the purchase library (eWebClient.lib), you can use eWeb_Today. See eWeb_Today.

Required Files

  • validate.h
    Included in the development project.
  • validateLibrary.lib
    Included in the development project.

Definition

eSellerate_DaysSince2000 eSellerate_Today ()

Example

#include "validate.h"

eSellerate_DaysSince2000 timeStamp;

timeStamp = eSellerate_DaysSince2000 eSellerate_Today()
