eWeb_WebStoreURL

Retrieves an affiliate-specific URL stored on the user's machine by eSellerate SalesTrac Technology (eST). See "eST downloads" in the eSellerate User's GuidePublishers Help index.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.

Definition

eWeb_WebStoreURL ( publisherID, skuRefNum, resultBuffer, resultBufferSize )

publisherID – Publisher ID string

skuRefNum – SKU RefNum string

resultBuffer – character buffer to hold URL string

resultBufferSize – size of character buffer


See eSellerate Return Values for details on return values.

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

skuRefNum

String containing the SKU RefNum, as found on the Sales Manager Available SKU Detail page. Required.

resultBuffer

Character buffer that eWeb_WebStoreURL will fill in with the URL it retrieves. Required.

resultBufferSize

Size of character buffer provided as resultBuffer . Must be large enough to hold the URL retrieved. Required.

Parameters for eWeb_WebStoreURL Function

char webStoreURL[256];

eSellerate_ArraySize webStoreURLSize =

eWeb_webStoreURL("PUB775527079", "SKU123456789", webStoreURL, 256);
