IsSystemCompatible

The IsSystemCompatible function determines whether the operating system will support the Embedded Web Store Engine. In the event that the function determines the system is incompatible, we recommend launching a standard eSellerate Web Store to allow your customers to complete their purchases.

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.

Definition

IsSystemCompatible () as Boolean

Example

dim eSellerate as eStore

dim resultCode as Boolean

eSellerate = new eStore

resultCode = eSellerate.IsSystemCompatible()

' ** TO DO: if error, launch standard eSellerate Web Store; otherwise use

' eSellerate Embedded Web Store **
