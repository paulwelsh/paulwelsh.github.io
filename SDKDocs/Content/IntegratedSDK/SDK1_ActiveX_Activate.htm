ActivateSerialNumber

For eSellerate Product Activation purposes, confirms that a serial number originated from eSellerate and associates that serial number with the unique hardware identification of the computer in use.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

ActivateSerialNumber (

publisherID As String, – Publisher ID string

activationID As String, – Product Activation Set ID string

serialNumber As String, – serial number string

askToConnect As Long – ask to connect when no active connection is found, and optionally display Manual Product Activation Wizard if connection attempt fails

) As Long

Return Value

Description

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if publisherID, activationID or serialNumber is "" or if the eSellerateEngine is unable to connect to the Internet. When no active connection to the Internet is found, the engine will return -1 if the value of the askToConnect parameter is false (zero) or if the value is true (non-zero) and the user cancels the dialog that asks for permission to connect. In addition, the engine will return -1 if the Manual Product Activation Wizard is displayed, and the user cancels.

0

Returned if the eSellerateEngine completes the action.

-25000

Returned if activation fails because activationID was unknown.

-25001

Returned if activation fails because serialNumber was unknown.

-25003

Returned if activation fails because serialNumber is blacklisted.

-25004

Returned if activation fails because the product was returned.

-25005

Returned if activation fails because the activation limit was met.

-25009

Returned if activation fails because serialNumber is already associated with another order.

-25012

Returned if eSellerate denies the activation request.
Return Values for ActivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Also controls whether the Manual Product Activation Wizard is displayed after the user grants permission to connect to the Internet but no connection can be made. Pass a true (non-zero other than 2) value to enable display of both the connection dialog and the wizard. Pass a value of 2 to enable display of the connection dialog but not the wizard. Pass false (zero) to disable display of both the connection dialog and the wizard.
Parameters for ActivateSerialNumber Function

Caution: Serial numbers used in Product Activation are case sensitive. This restriction is typically not an issue for SDK functions that handle serial numbers without user input, but it can be an issue if you use the ActivateSerialNumber function to allow customers to activate software purchased through a web store. To help prevent -25001 errors in this instance, you can configure your source code to correct any case discrepancies before passing a serialNumber parameter for the ActivateSerialNumber function. Also, in the prompt for your customers to enter the serial number, you can include language such as: "Enter the serial number exactly as shown on your receipt, including case (uppercase or lowercase)."

Example

Dim eSellerate As eSeller

Dim serialNumber As String

Dim resultCode As Long

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: obtain serialNumber **

resultCode = eSellerate.ActivateSerialNumber("PUB775527079", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU", serialNumber, 0)

' ** TO DO: handle error, or react to success **

Set eSellerate = Nothing
