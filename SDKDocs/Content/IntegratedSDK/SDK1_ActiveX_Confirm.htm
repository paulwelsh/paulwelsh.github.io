ConfirmSerialNumber

Confirms that a serial number is valid within the eSellerate system. A serial number is invalid if it is unknown, blacklisted or associated with an invalid order.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

ConfirmSerialNumber (

publisherID As String, – Publisher ID string

serialNumber As String, – serial number string

askToConnect As Long – ask to connect when no active connection is found

) As Long

Return Value

Description

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if publisherID or serialNumber is "" or if the eSellerateEngine is unable to connect to the Internet. When no active connection to the Internet is found, the engine will return -1 if the value of the AskToConnect parameter is false (zero) or if the value is true (non-zero) and the user cancels the dialog that asks for permission to connect.

0

Returned if the eSellerateEngine completes the action.

-25001

Returned if confirmation fails because serialNumber was unknown.

-25003

Returned if confirmation fails because serialNumber is blacklisted.

-25004

Returned if confirmation fails because serialNumber is associated with an invalid order.
Return Values for ConfirmSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

serialNumber

String containing the serial number. Required.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a true (non-zero) value to enable display of the dialog; otherwise pass false (zero).
Parameters for ConfirmSerialNumber Function

Note: Because some types of serial number sets permit duplicate serial numbers, ConfirmSerialNumber does not check to see if a serial number is associated with more than one order.

Example

Dim eSellerate As eSeller

Dim serialNumber As String

Dim resultCode As Long

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: obtain serialNumber **

resultCode = eSellerate.ConfirmSerialNumber("PUB775527079", serialNumber, 0)

' ** TO DO: handle error, or react to success **

Set eSellerate = Nothing
