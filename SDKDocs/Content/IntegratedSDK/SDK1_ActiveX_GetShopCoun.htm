GetShoppingCartCount

Organizes the shopping cart information obtained by a Wizard-invoking function and provides a count of the items in the shopping cart.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Definition

GetShoppingCartCount As Long

Return Value

Description

-1

Returned if the Wizard-invoking function failed to complete.

0

Returned if the Wizard-invoking function completed, but no items were in the shopping cart..

n

Returned if the Wizard-invoking function completed, and n items were in the shopping cart.

Return Values for GetShoppingCartCount Function

Note: A Wizard-invoking function must be called before GetShoppingCartCount is called.

Call GetShoppingCartCount to organize data only once after calling a Wizard-invoking function.

Example

Dim eSellerate As eSeller

Dim shoppingCartCount As Long

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: call a Wizard-invoking function **

shoppingCartCount = eSellerate.GetShoppingCartCount

' ** TO DO: handle error, or utilize results **

Set eSellerate = Nothing
