StandardizeSerialNumber

Takes a Standard or Standard Enhanced serial number that has been validated with ValidateSerialNumber and prepares the serial number for use with ActivateSerialNumber.

This function bridges the gap between ValidateSerialNumber, which tolerates certain characters if they are mistakenly entered in Standard or Standard Enhanced serial numbers, and ActivateSerialNumber, which requires the exact serial numbers.

To prepare a validated serial number for activation, StandardizeSerialNumber replaces certain characters that should never appear in a Standard or Standard Enhanced serial number (see the ValidateSerialNumber notes), substitutes uppercase characters for lowercase ones and writes the result to a string.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory
