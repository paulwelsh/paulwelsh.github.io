eSellerate_SilentCheckForUpdate

Checks for an update using the eSellerateEngine without invoking the Software Delivery Wizard.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib(Carbon) or
    libeSellerate.a (Mach-O version)
    Included in the development project.
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision

Definition

eSellerate_ErrorCode eSellerate_SilentCheckForUpdate ( publisherID, esellerID, previewID, askToConnect, trackingID )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString esellerID – Integrated eSeller RefNum string

eSellerate_InputString previewID – Preview Certificate string

eSellerate_TrueOrFalse askToConnect – ask to connect when no active connection is found

eSellerate_InputString trackingID – reserved for future functionality

Return Value

Description

eSellerate_FAILURE

Returned if publisherID or esellerID is nil or if no active connection to the Internet is found and the value of the askToConnect parameter is false (zero).

eSellerate_SUCCESS

Returned if an update is available.

eSellerate_NOUPDATE

Returned if no update is available.
Return Values for eSellerate_SilentCheckForUpdate Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

esellerID

String containing the Integrated eSeller RefNum, as found on the Sales Manager Integrated eSeller Summary page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Integrated eSeller Summary page. Required to purchase in Preview mode. Pass nil in Active mode.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a true (non-zero) value to enable display of the dialog; otherwise pass false (zero).

trackingID

String reserved for future functionality. Pass nil.

Type Definitions for eSellerate_SilentCheckForUpdate Function

Note: Strings are handled as length-prepended Pascal strings.

Caution: The eSellerate_SilentCheckForUpdate function cannot detect an active Internet connection on computers that are using an AirPort wireless network card. As a result, unless the function call prompts the user for permission to connect, the return value will be eSellerate_FAILURE. You can work around this limitation by passing a true (non-zero) value for the askToConnect parameter, which will enable display of the connection dialog.

Example

#include "eSellerate.h"

eSellerate_ErrorCode resultCode = eSellerate_SilentCheckForUpdate("\pPUB775527079", "\pES123456789", nil, 0, nil);
