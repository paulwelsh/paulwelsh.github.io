DeactivateSerialNumber

For eSellerate Product Activation purposes, removes the association between a serial number and the unique hardware identification of the computer in use.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision
Windows Targets
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

DeactivateSerialNumber (

publisherID As String, – Publisher ID string

activationID As String, – Product Activation Set ID string

serialNumber As String – serial number string

) as Long

Return Value

Description

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if publisherID, activationID or serialNumber is "" or if no active connection to the Internet is found and the value of the AskToConnect option is FALSE.

0

Returned if the eSellerateEngine completes the action.

-25001

Returned if deactivation fails because serialNumber was unknown.

-25003

Returned if deactivation fails because serialNumber is blacklisted.

-25004

Returned if deactivation fails because the product was returned.

-25006

Returned if deactivation fails because the activation key was unknown.

-25007

Returned if deactivation fails because the activation key was not the one associated with serialNumber.

-25008

Returned if deactivation fails because the activation key was invalid.

-25009

Returned if deactivation fails because serialNumber is

already associated with another order.

-25010

Returned if the activation key is valid, but deactivation fails because the key was not found on the eSellerate servers.

-25011

Returned if deactivation fails because the deactivation limit was met.
Return Values for DeactivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

Parameters for DeactivateSerialNumber Function

Option Name

Description

"AskToConnect"

String that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass "TRUE" to enable display of the dialog; otherwise pass "FALSE".
Options for DectivateSerialNumber Function

Options are set by SetOption. See SetOption.
