GetFileLocationID

Retrieves the Product ID of a file downloaded by a Wizard-invoking function from the information organized by GetFileLocationCount.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.

Definition

GetFileLocationID (

index as Integer – file number

) as String

Return Value

Description

""

Returned if there is no corresponding downloaded file.

file location ID

Returned if there is a corresponding downloaded file, where file location ID is the Product ID of that file.
Return Values for GetFileLocationID Function

Parameter

Description

index

File number. Indexing runs from 0 up to one less than the count returned by GetFileLocationCount. Required.
Parameters for GetFileLocationID Function

Note: GetFileLocationCount must be called before GetFileLocationID is called.

Example

dim eSellerate as eSeller

dim productIDArray(64) as String

dim count, index as Integer

eSellerate = new eSeller

' ** TO DO: call a Wizard-invoking function **

count = eSellerate.GetFileLocationCount - 1

for index = 0 to count

productIDArray(index) = eSellerate.GetFileLocationID(index)

end

' ** TO DO: utilize productIDArray
