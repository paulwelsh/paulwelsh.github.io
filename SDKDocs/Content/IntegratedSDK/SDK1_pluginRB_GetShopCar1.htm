GetShoppingCartSKU

Retrieves the SKU ID of an item obtained by a Wizard-invoking function from the information organized by GetShoppingCartCount.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.

Definition

GetShoppingCartSKU (

index as Integer – item number

) as String

Return Value

Description

""

Returned if there is no corresponding shopping cart item.

shopping cart SKU

Returned if there is a corresponding shopping cart item, where shopping cart SKU is the SKU ID of that item.
Return Values for GetShoppingCartSKU Function

Parameter

Description

index

Item number. Indexing runs from 0 up to one less than the count returned by GetShoppingCartCount. Required.
Parameters for GetShoppingCartSKU Function

Note: GetShoppingCartCount must be called before GetShoppingCartSKU is called.

Example

dim eSellerate as eSeller

dim skuIDArray(64) as String

dim count, index as Integer

eSellerate = new eSeller

' ** TO DO: call a Wizard-invoking function **

count = eSellerate.GetShoppingCartCount - 1

for index = 0 to count

skuIDArray(index) = eSellerate.GetShoppingCartSKU(index)

end
