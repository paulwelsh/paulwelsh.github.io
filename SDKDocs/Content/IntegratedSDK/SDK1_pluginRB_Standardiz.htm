StandardizeSerialNumber

Takes a Standard or Standard Enhanced serial number that has been validated with ValidateSerialNumber and prepares the serial number for use with ActivateSerialNumber.

This function bridges the gap between ValidateSerialNumber, which tolerates certain characters if they are mistakenly entered in Standard or Standard Enhanced serial numbers, and ActivateSerialNumber, which requires the exact serial numbers.

To prepare a validated serial number for activation, StandardizeSerialNumber replaces certain characters that should never appear in a Standard or Standard Enhanced serial number (see the ValidateSerialNumber notes), substitutes uppercase characters for lowercase ones and writes the result to output buffer correctedSerialNumber.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.

Definition

StandardizeSerialNumber (

validatedSerialNumber as String, – string containing the serial number

correctedSerialNumber as String – buffer for holding the corrected serial number string

)

Parameter

Description

validatedSerialNumber

String containing the Standard or Standard Enhanced serial number that has been validated by ValidateSerialNumber. Required.

correctedSerialNumber

Buffer that StandardizeSerialNumber will fill in with the corrected serial number. Required.
