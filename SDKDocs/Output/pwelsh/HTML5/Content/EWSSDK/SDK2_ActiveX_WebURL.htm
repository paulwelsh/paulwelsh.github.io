eWeb_WebStoreURL

Retrieves an affiliate-specific URL stored on the user's machine by eSellerate SalesTrac Technology (eST). See "eST downloads" in the eSellerate Publishers Help index.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at Installing eWebControl365.dll (the recommended location). It may be already placed here by another eSellerate-enabled application.

Definition

eWeb_WebStoreURL ( publisherID, skuRefNum, resultBuffer )

publisherID – Publisher ID string

skuRefNum – SKU RefNum string

resultBuffer – character buffer to hold URL string


See eSellerate Return Values for details on return values.

Parameters for eWeb_WebStoreURL Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

skuRefNum

String containing the SKU RefNum, as found on the Sales Manager Available SKU Detail page. Required.

resultBuffer

Character buffer that eWeb_WebStoreURL will fill in with the URL it retrieves. Required.

Example

Dim esellerate As eWebSDKLib.eWebSDK

Dim result As eWebSDKLib.eWeb_ReturnValues

Dim webStoreURL As String

 

esellerate = CreateObject("eWebSDK.365")

 

result = esellerate.eWeb_WebStoreURL("PUB775527079", "SKU123456789", webStoreURL)
