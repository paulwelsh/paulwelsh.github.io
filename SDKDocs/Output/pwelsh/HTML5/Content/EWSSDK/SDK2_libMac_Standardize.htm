eWeb_StandardizeSerialNumber

Takes a Standard or Standard Enhanced serial number that has been validated with eWeb_Validate-SerialNumber and prepares the serial number for use with eWeb_ActivateSerialNumber.

This function bridges the gap between eWeb_ValidateSerialNumber, which tolerates certain characters if they are mistakenly entered in Standard or Standard Enhanced serial numbers, and eWeb_ActivateSerialNumber, which requires the exact serial numbers.

To prepare a validated serial number for activation, eWeb_StandardizeSerialNumber replaces certain characters that should never appear in a Standard or Standard Enhanced serial number (see the eWeb_ValidateSerialNumber notes), substitutes uppercase characters for lowercase ones and writes the result to output buffer correctedSerialNumber.

This function relies on the library libValidateUniversal.a, which you will need to link into your application.

Required Files

  • validateLib.h
    Included in the development project.
  • libValidateUniversal.a
    Linked into your application.

Definition

eWeb_StandardizeSerialNumber ( validatedSerialNumber, correctedSerialNumber )

validatedSerialNumber – string containing the serial number

correctedSerialNumber – buffer for holding the corrected serial number string

Parameters for eWeb_StandardizeSerialNumber Function

Parameter

Description

validatedSerialNumber

String containing the Standard or Standard Enhanced serial number that has been validated by eWeb_ValidateSerialNumber. Required.

correctedSerialNumber

Buffer that eWeb_StandardizeSerialNumber will fill in with the corrected serial number. Required.
