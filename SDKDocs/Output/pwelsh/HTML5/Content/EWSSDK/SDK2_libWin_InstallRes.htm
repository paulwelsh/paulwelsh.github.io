eWeb_InstallEngineFromResource

Copies the eSellerate Embedded Web Store Engine (eWebClient.dll) from resources specified to the proper location on the user's machine. Replaces an older version of the engine if one is present. See also Default Locations for eWebClient.dll.

This approach requires that the eSellerate Embedded Web Store Engine is built into the application as a resource. It allows the application to not depend on an installer or other strategy to ensure that the engine is in place.

For information on installing the eSellerate Embedded Web Store Engine with an installer instead of the eWeb_InstallEngineFromResource function, see Installing eWebClient.dll .

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • eWebCompress
    Included in the development project.

Definition

HRESULT eWeb_InstallEngineFromResource ( ownerModule, resourceID, resourceType )

ownerModule – handle to a valid Win32 module that contains the engine resource

resourceID – ID given to the engine resource within ownerModule

resourceType – type-name given to the engine resource within ownerModule


See eSellerate Return Values for details on return values.

Parameters for eWeb_InstallEngineFromResource Function

Parameter

Description

ownerModule

Handle to a valid Win32 module that contains the engine resource. Pass NULL if the application making the call contains the engine resource.

resourceID

The ID given to the engine resource within ownerModule. Required.

resourceType

The type-name given to the engine resource within ownerModule. Required.

Tip: For the best protection against engine tampering, you should compile your application with the latest eSellerate libraries, and include the latest eSellerate Embedded Web Store Engine as a resource. Next, use the API call eWeb_InstallEngineFromResource immediately before any call that requires the engine. Calling eWeb_InstallEngineFromResource replaces an altered engine, if one is present.

Example

#include "eWebLibrary.h"

HRESULT result = eWeb_InstallEngineFromResource(NULL, IDR_ENGINE1, "ENGINE");
