eSellerate_ValidateSerialNumber

Checks the validity of a Standard or Standard Enhanced serial number.

Required Files

  • validate.h
    Included in the development project.
  • validateLibrary.lib
    Included in the development project.

Definition

eSellerate_DaysSince2000 eSellerate_ValidateSerialNumber ( serialNumber, nameBasedKey, extraDataKey, publisherKey )

serialNumber – serial number string

nameBasedKey – Name-based Key string

extraDataKey – additional validation key string

publisherKey – Publisher Key string

Tip: This function returns 0 if serialNumber is invalid. For a valid serialNumber, this function returns n, where n is a date expressed as the number of days since Jan. 1, 2000. If Duration is set on the Sales Manager Serial Number Set Detail page, then n is the date of expiration; otherwise, it is the date of purchase.

Parameters for eSellerate_ValidateSerialNumber Function

Parameter

Description

serialNumber

String containing the serial number. Required.

nameBasedKey

String containing the Name-based Key to be used to validate serialNumber. If the Based On setting on the Sales Manager Serial Number Set Detail page is set as None, pass NULL.

extraDataKey

String containing additional information to be used to validate serialNumber. If your Embedded Web Store eSeller did not use a special parameter (extraData) to pass additional information to the eSellerate servers during an API call to eWeb_Purchase or eWeb_CheckForUpdate, pass NULL.

publisherKey

String containing the Publisher Key to be used to validate serialNumber, as found on the Sales Manager Serial Number Set Detail page. Required.

Caution: The publisherKey is not the same as publisherID used in functions that invoke the Embedded Web Store.

Note: For the key portion of a Standard or Standard Enhanced serial number (see "serial numbers: Serial Number Type options" in the eSellerate User's Guide index), eSellerate does not use the letters I, O, U or Z because they can be easily confused with other characters. To accommodate customers who mistakenly enter these letters in Standard or Standard Enhanced serial numbers, eSellerate_ValidateSerialNumber will substitute the number 1 for I, the number 0 for O, the letter V for U and the number 2 for Z. The function also will substitute uppercase characters for lowercase ones.
