ActivateSerialNumber

Activates a serial number for use on the current host machine using eSellerate Product Activation.

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.
Macintosh targets
  • EWSMac.framework
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.
Windows targets
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

ActivateSerialNumber ( publisherID, activationID, serialNumber ) as Integer

publisherID as String – Publisher ID string

activationID as String – Product Activation Set ID string

serialNumber as String – serial number string


See eSellerate Return Values for details on return values.

Parameters for ActivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate Publishers Help index.

serialNumber

String containing the serial number. Required.
Options for ActivateSerialNumber Function

Option

Description

"AskToConnect"

Logical value that controls whether the Manual Product Activation Wizard is displayed if no active connection to the Internet is found. Pass a TRUE value to enable display of the wizard; otherwise pass FALSE.

Note: Options are set by SetOption. See SetOption.

Caution: Serial numbers used in Product Activation are case sensitive. This restriction is typically not an issue for SDK functions that handle serial numbers without user input, but it can be an issue if you use the ActivateSerialNumber function to allow customers to activate software purchased through a web store. To help prevent -25001 errors in this instance, you can configure your source code to correct any case discrepancies before passing a serialNumber parameter for the ActivateSerialNumber function. Also, in the prompt for your customers to enter the serial number, you can include language such as: "Enter the serial number exactly as shown on your receipt, including case (uppercase or lowercase)."

A related precaution that you can take to help prevent errors is to remove any leading and trailing spaces from the user input before passing the serialNumber parameter.

Example

dim eSellerate as eStore

dim serialNumber as String

dim resultCode as Integer

eSellerate = new eStore

' ** TO DO: obtain serialNumber **

resultCode = eSellerate.ActivateSerialNumber("PUB775527079", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU", serialNumber)

' ** TO DO: handle error, or react to success **
