GetOrderCount

When called after an Embedded Web Store invoking function, GetOrderCount provides a count of the order items. The result can then be used with GetOrderItemAttribute to extract information about a single order item.

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.
Macintosh targets
  • EWSMac.framework
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.
Windows targets
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

GetOrderCount () as Integer


See eSellerate Return Values for details on return values.

Example

dim eSellerate as eStore

dim orderCount as Integer

eSellerate = new eStore

' ** TO DO: call an Embedded Web Store invoking function **

orderCount = eSellerate.GetOrderCount()

' ** TO DO: use result to extract information about a single order item using

' GetOrderItemAttribute **
