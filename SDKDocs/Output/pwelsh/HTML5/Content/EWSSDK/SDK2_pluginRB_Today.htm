Today

Returns the current date (in days since 2000).

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.

Definition

Today () as Integer

Example

dim eSellerate as eStore

dim resultCode as Integer

eSellerate= new eStore

resultCode = eSellerate.Today()
