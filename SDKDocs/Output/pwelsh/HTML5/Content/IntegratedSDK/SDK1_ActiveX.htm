eSellerate ActiveX Control

Note: The eSellerate (Integrated) SDK is a legacy option that was superseded by the Embedded Web Store SDK. It is not recommended for use with new projects. The Embedded Web Store SDK provides many advanced features that are not available with the legacy option, including fully customizable stores and universal binary support for Macs.

This section details the eSellerate SDK provided by the eSellerate ActiveX control option for eSeller integration into Windows applications developed using Visual Basic or other COM-compatible tool. It documents the methods of the eSeller class:

For a summary of previous and current function calls applicable to the eSellerate ActiveX control option, see New Functions Summary.

Note: Beginning with the version 3.6.5 SDK, the recommended install locations for eSellerateControl365.dll and eSellerateEngine.dll are different for Vista than other Windows versions. The recommended locations for both DLLs are the ProgramData\eSellerate directory on Vista, and the Windows directory on pre-Vista Windows. As with previous SDK versions, you will need to register eSellerateControl.dll after installing it.
