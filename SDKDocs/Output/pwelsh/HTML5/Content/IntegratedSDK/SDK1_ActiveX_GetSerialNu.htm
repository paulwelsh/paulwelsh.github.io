GetSerialNumber

Retrieves a serial number from the serial number information obtained by a Wizard-invoking function and organized by GetSerialNumberCount.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Return Values for GetSerialNumber Function

Return Value

Description

""

Returned if no serial number for the item specified is available.

serial number

Returned if a serial number for the item specified is available, where that serial number is the string serial number.
Parameters for GetSerialNumber Function

Parameter

Description

index

Item number. Indexing runs from 1 up to the count returned by GetSerialNumberCount. Required.

Note: GetSerialNumberCount must be called before GetSerialNumber is called.

Example

Dim eSellerate As eSeller

Dim serialNumber As String

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: call a Wizard-invoking function **

If eSellerate.GetSerialNumberCount > 0 Then

serialNumber = eSellerate.GetSerialNumber(1)

' ** TO DO: utilize serialNumber **

End If

Set eSellerate = Nothing
