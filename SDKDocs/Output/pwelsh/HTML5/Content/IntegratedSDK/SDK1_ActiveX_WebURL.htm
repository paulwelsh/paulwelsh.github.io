GetWebStoreURL

Retrieves a web store URL stored by eSellerate SalesTrac Technology (eST) functions. See "eST downloads" in the eSellerate Publishers Help index.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Definition

GetWebStoreURL (

publisherID As String, – Publisher ID string

skuRefNum As String – SKU RefNum string

) As String

Return Values for GetWebStoreURL Function

Return Value

Description

""

Returned if no URL was found.

web store URL

Returned if the URL was found.
Parameters for GetWebStoreURL Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

skuRefNum

String containing the SKU RefNum, as found on the Sales Manager Available SKU Detail page. Required.

Example

Dim eSellerate As eSeller

Dim webStoreURL As String

Set eSellerate = CreateObject("eSellerateControl.365")

webStoreURL = eSellerate.GetWebStoreURL("PUB775527079", "SKU123456789")
