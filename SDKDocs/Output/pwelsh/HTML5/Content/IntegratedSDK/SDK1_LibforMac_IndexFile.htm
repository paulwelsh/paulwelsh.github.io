eSellerate_IndexFileLocationData

Builds access to locations of files downloaded by a Wizard-invoking function.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib(Carbon) or
    libeSellerate.a (Mach-O version)
    Included in the development project.

Definition

eSellerate_ArraySize eSellerate_IndexFileLocationData ( resultData, productIDArray, fileLocationArray, arraySize )

eSellerate_ResultData resultData – results obtained by the Software Delivery Wizard

eSellerate_OutputString[] productIDArray – array to list Product IDs corresponding to downloaded files

eSellerate_FileLocation[] fileLocationArray – array to list locations of downloaded files

eSellerate_ArraySize arraySize – number of elements in the arrays

Return Values for eSellerate_IndexFileLocationData Function

Return Value

Description

n

Number n of file locations listed to fileLocationArray. Bounded by arraySize.

Parameters for eSellerate_IndexFileLocationData Function

Parameter

Description

resultData

Results obtained by the Software Delivery Wizard. Required. The results are those obtained by calling Wizard-invoking functions.

productIDArray

Array that eSellerate_IndexFileLocationData will fill in to index Product ID strings held in resultData. The file location of an item identified by a Product ID is the fileLocationArray element with the same index as the ProductIDArray element. ProductIDArray may be nil if unneeded.

fileLocationArray

Array that eSellerate_IndexFileLocationData will fill in to index file location information held in resultData. Required.

arraySize

Number of elements in array provided as fileLocationArray. When productIDArray is provided as non-nil, it should have the same number of elements as fileLocationArray.

Note: Strings are handled as length-prepended Pascal strings.

Note: Call eSellerate_IndexFileLocationData to organize data only once after calling a Wizard-invoking function.

Example

#include "eSellerate.h"

eSellerate_ResultData resultData = nil;

eSellerate_FileLocation fileLocationArray[1];

eSellerate_ArraySize fileLocationCount;

/* TO DO: obtain resultData by calling a Wizard-invoking function */

fileLocationCount = eSellerate_IndexFileLocationData(resultData, nil, fileLocationArray, 1);

if ( fileLocationCount ) {

/* TO DO: utilize fileLocationArray */

}

eSellerate_DeleteResultData(resultData);
