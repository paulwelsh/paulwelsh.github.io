eSellerate_StandardizeSerialNumber

Takes a Standard or Standard Enhanced serial number that has been validated with eSellerate_ValidateSerialNumber and prepares the serial number for use with eSellerate_ActivateSerialNumber.

This function bridges the gap between eSellerate_ValidateSerialNumber, which tolerates certain characters if they are mistakenly entered in Standard or Standard Enhanced serial numbers, and eSellerate_ActivateSerialNumber, which requires the exact serial numbers.

To prepare a validated serial number for activation, eSellerate_StandardizeSerialNumber replaces certain characters that should never appear in a Standard or Standard Enhanced serial number (see the eSellerate_ValidateSerialNumber notes), substitutes uppercase characters for lowercase ones and writes the result to output buffer correctedSerialNumber.

Definition

eSellerate_StandardizeSerialNumber ( validated  SerialNumber, correctedSerialNumber )

eSellerate_String validatedSerialNumber – string containing the serial number

eSellerate_String correctedSerialNumber – buffer for holding the corrected serial number string

Parameters for eSellerate_StandardizeSerialNumber Function

Parameter

Description

validatedSerialNumber

String containing the Standard or Standard Enhanced serial number that has been validated by eSellerate_ValidateSerialNumber. Required.

correctedSerialNumber

Buffer that eSellerate_StandardizeSerialNumber will fill in with the corrected serial number. Required.
