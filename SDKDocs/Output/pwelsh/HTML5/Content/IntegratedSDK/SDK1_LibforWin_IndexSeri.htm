eSellerate_IndexSerialNumberData

Builds access to serial number information obtained by a Wizard-invoking function.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.

Definition

eSellerate_ArraySize eSellerate_IndexSerialNumberData ( resultData, productIDArray, serialNumberArray, nameBasedKeyArray, arraySize )

eSellerate_ResultData resultData – results obtained by the Software Delivery Wizard

eSellerate_OutputString[] productIDArray – array to list Product IDs corresponding to listed serial numbers

eSellerate_OutputString[] serialNumberArray – array to list serial numbers

eSellerate_OutputString[] nameBasedKeyArray – array to list Name-based Keys corresponding to listed serial numbers

eSellerate_ArraySize arraySize – number of elements in the arrays

Return Values for eSellerate_IndexSerialNumberData Function

Return Value

Description

n

Number n of serial numbers listed to serialNumberArray. Bounded by arraySize.

Parameters for eSellerate_IndexSerialNumberData Function

Parameter

Description

resultData

Results obtained by the Software Delivery Wizard. Required. The results are those obtained by calling Wizard-invoking functions.

productIDArray

Array that eSellerate_IndexSerialNumberData will fill in to index Product ID strings held in resultData. The serial number of an item identified by a Product ID is the serialNumberArray element with the same index as the ProductIDArray element. ProductIDArray may be NULL if unneeded.

serialNumberArray

Array that eSellerate_IndexSerialNumberData will fill in to index serial number strings held in resultData. Required.

nameBasedKeyArray

Array that eSellerate_IndexSerialNumberData will fill in to index Name-based Key strings held in resultData. The serial number that a Name-based Key is used with is the serialNumberArray element with the same index as the nameBasedKeyArray element. NameBasedKeyArray may be NULL if unneeded.

arraySize

Number of elements in array provided as serialNumberArray. When productIDArray or nameBasedKeyArray are provided as non-NULL, these arrays should have the same number of elements as serialNumberArray.

Type Definitions for eSellerate_IndexSerialNumberData Function

Data Type

Definition

eSellerate_ArraySize

signed short

eSellerate_ResultData

void*

eSellerate_OutputString

char*

Note: Strings are handled as zero-appended C strings.

Note: The Name-based Key for a serial number is always "" if the Based On setting on the Sales Manager Serial Number Set Detail page is set as None.

Note: Call eSellerate_IndexSerialNumberData to organize data only once after calling a Wizard-invoking function.

Example

#include "eSellerate.h"

eSellerate_ResultData resultData = NULL;

eSellerate_OutputString serialNumberArray[1];

eSellerate_OutputString nameBasedKeyArray[1];

eSellerate_ArraySize serialNumberCount;

/* TO DO: obtain resultData by calling a Wizard-invoking function */

serialNumberCount = eSellerate_IndexSerialNumberData(resultData, NULL, serialNumberArray, nameBasedKeyArray, 1);

if ( serialNumberCount ) {

/* TO DO: utilize serialNumberArray and nameBasedKeyArray */

}

eSellerate_DeleteResultData(resultData);
