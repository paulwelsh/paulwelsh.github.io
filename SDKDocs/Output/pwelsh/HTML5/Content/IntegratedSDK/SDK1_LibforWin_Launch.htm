eSellerate_LaunchUpdate

Launches a standalone executable, such as an installer, or applies an Updater VISE file to a specified file and launches the resulting updated file.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

eSellerate_ErrorCode eSellerate_LaunchUpdate ( updaterFile, fileToUpdate )

eSellerate_FileLocation updaterFile – location of an Updater VISE file to apply or an executable to launch

eSellerate_FileLocation fileToUpdate – location of the file that an Updater VISE file is to be applied to

Return Values for eSellerate_LaunchUpdate Function

Return Value

Description

eSellerate_NOENGINE

Returned if the eSellerateEngine cannot be found.

eSellerate_FAILURE

Returned if updaterFile cannot be launched.

eSellerate_SUCCESS

Returned if updaterFile does launch.

Parameters for eSellerate_LaunchUpdate Function

Parameter

Description

updaterFile

Location of either an Updater VISE file to apply to fileToUpdate or a standalone executable to launch. Required.

fileToUpdate

Location of the file to apply updaterFile to, if updaterFile is an Updater VISE file. If it is not, pass NULL to launch a standalone executable, such as an installer.

Type Definitions for eSellerate_LaunchUpdate Function

Data Type

Definition

eSellerate_ErrorCode

signed long

eSellerate_FileLocation

char*

Note: File locations are expressed as pathnames, and pathnames are handled as zero-appended C strings.

Note: It may well be the case that a running application calls eSellerate_LaunchUpdate for the purpose of updating itself. In this situation, the application should promptly terminate itself after calling eSellerate_LaunchUpdate, since the application is then to be replaced.

Example

#include "eSellerate.h"

eSellerate_FileLocation updaterFile;

eSellerate_FileLocation fileToUpdate;

eSellerate_ErrorCode resultCode;

/* TO DO: obtain updaterFile and, if needed, fileToUpdate */

eSellerate_resultCode = eSellerate_LaunchUpdate(updaterFile, fileToUpdate);

/* TO DO: if fileToUpdate is the running application, exit application */
