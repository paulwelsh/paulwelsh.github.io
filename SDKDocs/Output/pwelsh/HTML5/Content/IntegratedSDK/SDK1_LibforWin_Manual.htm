eSellerate_ManualActivateSerialNumber

For eSellerate Product Activation purposes, enables activation when the computer's configuration prevents the eSellerateEngine from accessing the eSellerate servers. Invokes a wizard and allows the user to manually activate the software using a web browser on the current computer or another computer with web access. Otherwise works like the eSellerate_ActivateSerialNumber function in that it confirms that a serial number originated from eSellerate and associates that serial number with the unique hardware identification of the computer in use.

Also see "Product Activation: manually activating serial numbers" in the eSellerate Publishers Help index.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

eSellerate_ErrorCode eSellerate_ManualActivateSerialNumber ( publisherID, activationID, serialNumber, titlePreface, otherOptions )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString activationID – Product Activation Set ID string

eSellerate_InputString serialNumber – serial number string

eSellerate_InputString titlePreface – custom text, to preface activation wizard titles

eSellerate_InputString otherOptions – custom text, to include phone number, e-mail or other contact

Return Values for eSellerate_ManualActivateSerialNumber Function

Return Value

Description

eSellerate_NOENGINE

Returned if the eSellerateEngine cannot be found.

eSellerate_FAILURE

Returned if publisherID, activationID or serialNumber is NULL or if the eSellerateEngine does not complete the action (such as when the user cancels the Manual Product Activation Wizard).

eSellerate_SUCCESS

Returned if the eSellerateEngine completes the action.

Parameters for eSellerate_ManualActivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate Publishers Help index.

serialNumber

String containing the serial number. Required.

titlePreface

String containing a custom preface to the titles of all Manual Product Activation Wizard pages. Causes the titles to be "<Preface>: Product Activation." May be NULL, in which case the titles will simply be "Product Activation."

otherOptions

String containing custom information to help users without web access activate their software. (This might be contact information for the publisher, who would visit http://activate.esellerate.net to obtain an Activation Key for the user.) The wizard will display up to six lines for this string. May be NULL, in which case the option to activate without web access will not be available to users.
Type Definitions for eSellerate_ManualActivateSerialNumber Function

Data Type

Definition

eSellerate_ErrorCode

signed long

eSellerate_InputString

const char*

Note: Strings are handled as zero-appended C strings.

Note: Unless there is a specific need for your application source code to directly invoke the Manual Product Activation Wizard, it is more efficient to use the eSellerate_ActivateSerialNumber function instead of eSellerate_ManualActivateSerialNumber.

When used with the proper askToConnect parameter, eSellerate_ActivateSerialNumber can handle normal (automatic) activations and also invoke the Manual Product Activation Wizard when needed. In contrast, eSellerate_ManualActivateSerialNumber can only handle manual activations, and consequently is limited to use in special circumstances.

eSellerate_InputString serialNumber;

eSellerate_ErrorCode resultCode;

/* TO DO: obtain serialNumber */
