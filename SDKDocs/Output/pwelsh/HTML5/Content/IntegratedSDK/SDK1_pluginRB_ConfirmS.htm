ConfirmSerialNumber

Confirms that a serial number is valid within the eSellerate system. A serial number is invalid if it is unknown, blacklisted or associated with an invalid order.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision
Windows Targets
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

ConfirmSerialNumber (

publisherID As String, – Publisher ID string

serialNumber As String – serial number string

) as Long

Return Values for ConfirmSerialNumber Function

Return Value

Description

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if publisherID or serialNumber is "" or if the eSellerateEngine is unable to connect to the Internet. When no active connection to the Internet is found, the engine will return -1 if the value of the AskToConnect option is FALSE or if the value is TRUE and the user cancels the dialog that asks for permission to connect.

0

Returned if the eSellerateEngine completes the action.

-25001

Returned if confirmation fails because serialNumber was unknown.

-25003

Returned if confirmation fails because serialNumber is blacklisted.

-25004

Returned if confirmation fails because serialNumber is associated with an invalid order.
Parameters for ConfirmSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

serialNumber

String containing the serial number. Required.

Options for ConfirmSerialNumber Function

Option Name

Description

"AskToConnect"

String that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass "TRUE" to enable display of the dialog; otherwise pass "FALSE".

Note: Options are set by SetOption. See SetOption.

Note: Because some types of serial number sets permit duplicate serial numbers, ConfirmSerialNumber does not check to see if a serial number is associated with more than one order.
