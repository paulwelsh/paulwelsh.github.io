GetSerialNumber

Retrieves a serial number from the serial number information obtained by a Wizard-invoking function and organized by GetSerialNumberCount.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.

Definition

Return Values for GetSerialNumber Function

Return Value

Description

""

Returned if no serial number for the item specified is available.

serial number

Returned if a serial number for the item specified is available, where that serial number is the string serial number.
Parameters for GetSerialNumber Function

Parameter

Description

index

Item number. Indexing runs from 0 up to one less than the count returned by GetSerialNumberCount. Required.

Note: GetSerialNumberCount must be called before GetSerialNumber is called.

dim serialNumber as String
