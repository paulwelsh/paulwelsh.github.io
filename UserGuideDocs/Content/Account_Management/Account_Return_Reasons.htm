Creating Return Reasons

By default, the system offers the following selections for use in processing returned orders: Changed Mind; Defective; Other; Purchased Competitor's. It also allows you to create your own return reasons to supplement these default selections.

To create a return reason:
  1. Click the Return Reasons tab. The Return Reasons page appears.
  2. Click New. The Return Reason Detail page appears.
  3. In the Return Code field, enter text for identifying this item in searches.
  4. In the Description field, enter text for identifying this item in lists of available return reasons.
  5. If you're ready to make this return reason available for use, select the Active check box. Otherwise, you can save this return reason and activate it later.
  6. Click Save Changes.
