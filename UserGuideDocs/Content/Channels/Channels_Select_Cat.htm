Channels: Select SKU Channel Categories

You can use the dialog on this page to assign channel categories to the selected SKU. The categories determine where the SKU appears within the channel web site.

Each SKU must have at least one channel category assigned to it. The maximum is six categories per SKU.

Note: This dialog allows selecting multiple categories in a column at once. To select consecutive categories, hold down the Shift key and click on the desired items. To select non-consecutive categories, hold down the Control key (Windows) or the Command key (Macintosh) and click on the desired items.

To assign a channel category to the SKU, do the following:

  1. Select the category in the "Available Categories" column.
  2. Click Add.

To remove a channel category assignment, do the following:

  1. Select the category in the "Selected Categories" column.
  2. Click Remove.

When you are finished modifying the SKU's channel categories, click Save Changes.
