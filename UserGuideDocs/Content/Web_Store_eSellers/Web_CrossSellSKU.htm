Web Store SKUs: Cross-Sell SKU Group

When a web store customer adds this SKU to the shopping cart, you can offer other SKUs for sale in conjunction with the purchase. To set up this cross-selling relationship, you would assign a cross-sell SKU group to this SKU.

Note: You can only offer cross-sell SKUs that belong to this Web Store eSeller.

Click the Cross-Sell SKU Group list to select a group, or click Edit Cross-Sell SKU Groups to edit or create one. Otherwise, select None to skip this option.
