Custom Pixel Tracking Methods

With this option, you provide the URL for a tracking pixel, and eSellerate will inject that tracking pixel into your store pages.

To create a Custom Pixel tracking method:
  1. From the Tracking Method Type list, select Custom Pixel.
  2. In the Name field, enter a name for this tracking method. A name is not required, but it will help you more quickly identify this tracking method when it appears in lists (either on the Third-Party Tracking page or at the eSeller level).
  3. In the Description field, enter any information that you would like to note for this tracking method (optional).
  4. In the Tracking Pixel URL field, enter the URL for your tracking pixel.
  5. If you are ready to start using this tracking code in all of your stores, select the Enabled check box. Otherwise, you can leave this tracking method disabled for the time being. (See also Disabling Tracking Methods for Individual Stores.)
