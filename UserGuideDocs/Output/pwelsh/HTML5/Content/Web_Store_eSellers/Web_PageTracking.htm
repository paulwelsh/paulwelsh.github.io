Web Store: Page Tracking

The Page Tracking section of the Web Store eSeller Detail page includes the options described here.

From this section, you can create or edit tracking methods for this store. You also can edit or delete tracking methods for all stores. Information on how to complete some of the more typical tasks follows.

Creating a Tracking Method for This Store Only

  1. Click New. Additional tracking options appear.
  2. Make a selection from the Tracking Method Type list, and then fill out the settings for that tracking method. Help for the different types of tracking methods is available at:
    • Google Analytics Tracking Methods
    • Custom JavaScript Tracking Methods
    • Custom Pixel Tracking Methods
  3. Click Create.

Disabling a Tracking Method for This Store Only

  1. Select the tracking method that you want to disable for this store.
  2. Click Edit. Additional tracking options appear.
  3. Clear the Enabled check box.
  4. Click Save.

Editing Tracking Methods That Apply to All Stores

  1. Select the tracking method that you want to edit. (It should be one that shows "All Stores"—instead of the eSeller ID—in the Store column.)
  2. Click Edit. Additional tracking options appear.

  3. Make your changes.

    Note: The changes that you make here will affect all stores, with one exception: Clearing the Enabled check box will only affect the store that you are currently editing.

  4. Click Save.
