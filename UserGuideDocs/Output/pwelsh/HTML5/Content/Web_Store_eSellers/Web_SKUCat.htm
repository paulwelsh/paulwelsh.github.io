Web Store SKUs: SKU Category

The Web Store Category list is for organizing the display of SKUs in your web store. These categories will display in alphabetical order by default, or you can set the display order from the Web Store eSeller Summary page.

Note: Web Store SKUs that use no category will always appear last in your web store.

Click the Web Store Category list to select a category, or click Edit Categories to edit or create one. Otherwise, select No Category to skip this option.
