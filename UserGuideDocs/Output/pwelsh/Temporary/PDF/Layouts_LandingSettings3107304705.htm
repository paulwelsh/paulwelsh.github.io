Settings Tab

You can use the Settings tab of the Landing Page Detail page to enter Landing Page Name and Comment, as described here.

Note: When you're finished making changes to the Settings tab, be sure to click Update Settings.

Entering a Landing Page Name

Landing Page Name will be visible to your affiliates if you assign this Landing Page to SKUs that you list on the eSellerate Affiliates web site. For that reason, you should always enter names that are suitable for public display.

Note: Duplicate Landing Page names are allowed. However, there is no built-in way to differentiate between duplicate names when you assign Landing Pages to your SKUs.

Entering a Comment

If desired, you can edit the default comment. Text that you enter here is for your reference only, and will not be visible to your affiliates.
