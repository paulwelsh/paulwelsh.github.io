Creating a Web Store eSeller

To create a Web Store eSeller:
  1. Go to http://www.esellerate.net and log in. The home page appears.
  2. Under SKU Management, click eSellers. The eSeller List page appears.
  3. To the right of Create New, click Web Store. The Web Store eSeller Detail page appears.
  4. Complete the Detail page settings as described at:
    • "Web Store: General Attributes" on page 1
    • "Web Store: Receipt Information" on page 1
    • "Web Store: Private Login" on page 1
    • "Web Store: Layout" on page 1
  5. Click Save.
