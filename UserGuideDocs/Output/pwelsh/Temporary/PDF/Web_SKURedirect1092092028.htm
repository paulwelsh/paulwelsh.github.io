Web Store SKUs: SKU Redirect

The SKU Redirect feature allows you to bypass the current Web Store SKU when you want to make a different SKU available for purchase. When a customer uses a direct "Buy" link to initiate a purchase of the current SKU, the SKU you specified in the SKU Redirect list will be made available instead.

Tip: For a look at all the SKU Redirects set up for your Web Store eSellers, use the SKU Redirects report, which is available in Admin Reports.
