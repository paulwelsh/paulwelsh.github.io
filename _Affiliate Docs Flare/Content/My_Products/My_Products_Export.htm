Exporting "My Products" Data

There are two ways to get your latest "My Products" data in XML format:

  • By using a web service
  • By exporting it from affiliates.esellerate.net as described below
Notes:
  • This option is for users who are already familiar with the instructions provided on the various tabs of the Product Detail page.
  • eSellerate provides the following resources to help you use the exported data:
    • The My Products Service WSDL, which also describes the data in version 2 of the data export: https://affiliates.esellerate.net/affiliates/ws/myproductsservice.asmx?wsdl
    • "My Products" XML Resources, a download that includes:
      • A sample XML document (Sample MyProducts.xml) that makes use of all the possible data fields
      • A Read Me file (MyProductsServiceReadMe.pdf) that is applicable only if you plan to use the web service

      The download is available at http://affiliates.esellerate.net/Affiliates/Downloads/MyProducts_XML_Resources.zip.

  • There are two settings on the Purchase Links tab that you cannot control when you export data: link appearance and link action. These settings will default to the first choice listed for each of them on the corresponding Purchase Links tab.
  • There is one setting on the Landing Pages tab that you cannot control when you export data: link appearance. This setting will default to the first choice listed for it on the corresponding Landing Pages tab.
  • The "My Products" data will note when the available options for selling a product include Downloadable eST, but it will not provide a link to the eST file. If you want to use the Downloadable eST option, you will need to go to the eST tab and follow the instructions provided.
  • PRODUCT_IMAGE_URL, BUY_URL, MORE_INFO_URL, and CATALOG_URL will be blank when SKU STATUS is Pending or Declined. The fields will be populated when SKU STATUS is Active.
  • eSellerate will be removing the version 1 export option by the end of 2009. If you use version 1, please make the transition to version 2 in advance of the cutoff.
To export your "My Products" data (version 1):
  1. If you want to export data for all products, you can accept the default export option, All. Then skip to step 4.
  2. Do one of the following to select the desired product(s):
    • To select individual products, check the box to the left of the product.
    • To select all products on the page, check the box in the table heading.
  3. From the Export popmenu, select Checked.
  4. If you specify a tracking ID, eSellerate will record that ID along with other order details whenever the "Buy" or "Try" links provided in the exported data generate a sale. You can then use the SKU Sales by Date Range report to find sales that involved the tracking ID. If you would like to use the tracking ID option, enter an ID in the field provided. The ID can be any alphanumeric name you choose, up to a 100-character limit.
  5. Click Export.
  6. Save the XML document.
To export your "My Products" data (version 2):
  1. If you want to export data for all products, you can accept the default export option, All. Then skip to step 4.
  2. From the Export popmenu, select one of the following options:
    • Checked
    • Active
    • Declined
    • Pending
  3. If you selected Checked in the preceding step, do one of the following to select the desired product(s):
    • To select individual products, check the box to the left of the product.
    • To select all products on the page, check the box in the table heading.
  4. If you specify a tracking ID, eSellerate will record that ID along with other order details whenever the "Buy" or "Try" links provided in the exported data generate a sale. You can then use the SKU Sales by Date Range report to find sales that involved the tracking ID. If you would like to use the tracking ID option, enter an ID in the field provided. The ID can be any alphanumeric name you choose, up to a 100-character limit.
  5. Click Export.
  6. Save the XML document.
