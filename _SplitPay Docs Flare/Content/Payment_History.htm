Payment History

Note: The information that is available through this report does not include your profit or eSellerate's commission on your orders from 4/1/2011 and later. To find that information and more, please use Business Navigator reports.

This report shows details on your previous payments.

To access Payment History report information:
  1. For more information about a specific payment, click its Payment ID field. The resulting page will include a publisher sales summary and a listing of any adjustment items.
  2. For more payment details, use one of the following Payment Transaction Ledger options:
    • Click View to see the results in an HTML table.
    • Click Export (CSV) to export the results as a comma-delimited file.
If you selected View:

The table will display all transactions involved in this payment.

If you selected Export (CSV):

The resulting Save As dialog will display a default file name, which you can change if desired. In any case, you will need to change the file extension to ".csv" before saving the file (applies to both Windows and Macintosh platforms).

You can now import the file into a database, or open it in a spreadsheet program such as Microsoft Excel. (If a program prompts you to specify a format for this data, select comma-delimited or comma-separated values.)
