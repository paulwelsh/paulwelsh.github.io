Configuring eCD

eCD is a CD fulfillment service available to publishers using Web Store and Embedded Web Store eSellers. With eCD, you can offer backup CDs as add-ons when customers purchase software from your web store. Or you can sell CDs as items that are always included with certain SKUs.

Overview

To get started with eCD, you will need to click through the About eCD page, which will enable eCD for your publisher account. See Getting Started: About eCD Page.

Next, review Notes on eCD.

You can then configure eCD, which involves the following steps:

  1. Organize the files that you want to provide on CD. See Organizing Your Files for eCD.
  2. Upload your files and optionally designate them for eCD use only. See Uploading eCD Files.
  3. Create custom images for various elements such as the CD face. See eCD Images.
  4. Create eCD Layouts, which control attributes such as the images used in eCD and the inclusion of any trial files. See eCD Layouts.
  5. Fill out the eCD Optional section of the eCD Configuration tab. This includes setting pricing and the layout for all SKUs that support eCD Optional. See eCD Configuration.
  6. Assign an eCD file to each SKU that should support eCD Optional. See Valid SKUs.
  7. Do the following for each SKU that should support eCD Included:
    • Add eCD into the SKU price. See Pricing and Creating the SKU and Setting Its Basic Properties.
    • Set the following properties: Delivery Method, eCD File, eCD Layout and, when needed, eCD Delivery Frequency. See Setting the SKU's Advanced Properties (Optional).
  8. Set the file action for eCD files that are Zip archives. See Valid eCD Files.
  9. Once you're finished with the preceding steps, we recommend ordering some test CDs to verify that everything is as you expected. Test CDs will be at your own expense.
