Channel Returns

In sales partnerships established through the Channels program, the channel is responsible for product returns. Customers that purchase your products through a channel are instructed to contact that channel if they need to request refunds. However, because customers may still contact you to request refunds on channel orders, the Sales Manager provides functionality for directing those requests to the channel.

Orders that were placed through a channel will appear under "Channel Orders Found" in the Returns search results. If you look up an order to return and find that it is a channel order, you can facilitate the return process by doing the following:

  1. Tell the customer that you will forward the return request to the channel.
  2. Within the Returns search results, click the order's Order Number field. The Channel Returns page displays.
  3. Follow the instructions provided to e-mail the channel a return request.
