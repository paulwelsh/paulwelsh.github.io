SKUs: Channel

Information on this page appears as follows:

  • Software Requirements
  • System Requirements
  • Trialware

Software Requirements

Information that you specify here will display for the SKU within the channel, under the heading "Software." If you mark an attribute as Not Applicable (N/A) or leave it blank, the channel will not display an entry for that attribute.

License Type – The options for this attribute are N/A, Full Version or Subscription. If you select Subscription, you will need to enter a duration, and select either Year(s), Month(s) or Day(s).

License Quantity – The options for this attribute are N/A, Site License or User License. If you select User License, you will need to enter the number of users that the license allows.

Distribution Media – The options for this attribute are Electronic, Electronic with Shippable Media or Shippable. The default setting is Electronic.

Other Software Requirements – Here you can enter up to 255 characters of additional information about software requirements. (Do not include HTML tags.) Some examples of information that you might communicate through this text field are:

  • Age recommendations
  • Other software required for using the product, as would be the case for plug-ins
  • Tools required for handling compressed files

System Requirements

Information that you specify here will display for the SKU within the channel, under the heading "System Requirements." If you leave an attribute blank, the channel will not display an entry for that attribute.

Operating System – Select the appropriate operating system(s).

Note: You must specify at least one Operating System before you submit this SKU for the channel to sell.

Min Processor Type – Here you can enter up to 255 characters about the minimum processor type. (Do not include HTML tags.)

Min RAM Size – Here you can enter the number of megabytes for the minimum RAM size (numeric values only).

Min Hard Drive Space – Here you can enter the number of megabytes for the minimum hard drive space (numeric values only).

Other System Requirements – Here you can enter up to 255 characters of additional information about system requirements. (Do not include HTML tags.) Some examples of information that you might communicate through this text field are:

  • Video RAM requirements
  • Any special hardware or accessories that are required, such as a CD burner or a game controller
  • Broadband connection recommended

Trialware

If you want the channel to offer a trial download for the SKU that you're editing, you can specify that download here.

Note: Depending on how you set up your application to enable full functionality after a purchase, it's possible that your trial download could be the same one that's delivered when customers purchase your SKU through the channel.

To select a trial download, use the Trial Download list. If you need to edit or create a download file before you make your selection, click Save, and then click the Edit Downloads icon.

Note: The "Buy URL" listed here is the one that your trial download should use when your customers initiate purchases from within the application.
