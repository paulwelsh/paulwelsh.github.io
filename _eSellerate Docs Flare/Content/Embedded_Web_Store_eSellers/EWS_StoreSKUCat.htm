Setting Display Order for Categories and SKUs

After you add SKUs to your Embedded Web Store eSeller, you can organize the SKUs by category, and control their display order in the Embedded Web Store. These settings are available in the Embedded Web Store SKU Categories section of the Embedded Web Store eSeller Summary page.

Setting Category Display Order

By default, the SKUs you add to this Embedded Web Store eSeller will display in alphabetical order according to category. To change the ordering for a specific category, enter a number (0-999) in its Display Order field and then click Update Display Order.

Categories with the same Display Order value will display in alphabetical order.

SKUs that use no category will appear under No Category on this page, and will always display last in the Embedded Web Store.

Setting SKU Display Order

By default, SKUs will display in alphabetical order within categories. To change the SKU ordering within a category, click the category name. The Web Store Category SKU List page will display. To change the ordering for a specific SKU, enter a number (0-999) in its Display Order field and then click Update Status.

SKUs with the same Display Order value will display in alphabetical order.

Tip: The increments you use for SKU display order should reserve enough space for future SKU entries to fit in when needed. For example, the Display Order values for SKUA, SKUB and SKUC could be 25, 50 and 75, respectively.
