Integrated eSeller: Cross-Sell SKUs

When a customer uses this Integrated eSeller to get your primary SKU, you can offer other SKUs for sale in conjunction with the purchase. To set up this cross-selling relationship, you would assign a cross-sell SKU group to this SKU.

Click the Cross-Sell SKU Group list to select a group, or click Edit Cross-Sell SKU Groups to edit or create one. Otherwise, select None to skip this option.
