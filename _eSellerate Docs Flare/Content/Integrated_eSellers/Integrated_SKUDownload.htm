Integrated eSeller SKUs: Download

In the Download section of the Integrated eSeller SKU Attributes page, you can specify the default file to download during the delivery of this SKU. Your choices are as follows:

  • Use no download.
  • Use the default SKU download. (This is the default setting.)
  • Use an existing download file other than the default SKU download.

The download file that you specify for this SKU will override the default SKU download, if the two files are different. This behavior enables a SKU to have an individual download for each eSeller that uses it.

Note: Integrated eSellers running on the Macintosh platform call on the Engine for Macintosh to download and deliver files. That engine automatically decodes MacBinary files that it downloads from the servers and saves to disk. For file formats other than MacBinary, the engine downloads and saves the files to disk without decoding or uncompressing them.

Tip: To delete a download file, use the Downloads editor (available in the References section of the Sales Manager).
