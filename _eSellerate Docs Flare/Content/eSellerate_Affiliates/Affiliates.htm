eSellerate Affiliates

The eSellerate Affiliates section of the Sales Manager is for accomplishing the following tasks:

  • Joining eSellerate Affiliates
  • Entering Affiliate Contact Information
  • Submitting SKUs
  • Creating eST Downloads
  • Testing eST Downloads
  • Updating eST Downloads
  • Customizing Online Stores for Select Affiliates
  • Making Affiliate Level Changes
