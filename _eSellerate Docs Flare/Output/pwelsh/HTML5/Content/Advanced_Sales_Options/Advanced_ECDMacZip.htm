Macintosh Zip

Macintosh Zip archives (.zip) will each be placed in their own subdirectory on the CD, and an autorun HTML menu of the CD contents will be automatically created. On Windows systems, the user will be able to click an "Install" button in the menu to open the corresponding file. The autorun functionality will only be available on Windows.

Note: You can use the Valid eCD Files feature on the eCD Configuration tab to control whether Zip archives are extracted before they're placed on the CD.
