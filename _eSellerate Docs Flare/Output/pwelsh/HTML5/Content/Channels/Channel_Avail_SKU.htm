Channels: Available Channel SKUs

The Available Channel SKUs page is the place for managing all the SKUs that you've accepted into your channel. You can use this page to do any of the following:

  • Search for SKUs
  • Sort SKU lists
  • Download SKU "Buy" files
  • Download SKU "Try" files
  • Categorize SKUs
  • Access SKU settings

More details on the functions provided by the Available Channel SKUs page follow.

Searching for SKUs

The top portion of the page allows searching by criteria such as publisher name and SKU availability.

Sorting SKU Lists

By default, the page lists SKUs by price in ascending order. To list SKUs by a different criterion, click the appropriate table heading, such as SKU or Publisher. Click the same heading a second time to change to descending order if desired.

Downloading SKU "Buy" Files

Most SKUs that you sell through your channel web site will have a "Buy" file, which is the file customers receive when they purchase the SKU. To download a SKU's "Buy" file for evaluation purposes, select the SKU and then click Download Buy File. If you attempt the download for a SKU that does not have a "Buy" file, a message will display.

Downloading SKU "Try" Files

Some SKUs that you sell through your channel web site will have a "Try" file, which is the file customers receive when they download a trial version of the SKU. An "X" in the "Try Qualified" column indicates that the SKU has a "Try" file, and that you have not yet marked the SKU as "Try Qualified."

To download a SKU's "Try" file for evaluation purposes, select the SKU and then click Download Try File. If you attempt the download for a SKU that does not have a "Try" file, a message will appear.

Categorizing SKUs

The page provides the following settings for categorizing SKUs:

  • Category
  • In Channel
  • Buy Qualified
  • Try Qualified

Category – The Category settings determine where the SKUs appear within the channel web site. These channel categories are separate from any other categories the publisher assigns to the SKUs within the Sales Manager.

Each SKU offered for sale in the channel web site must have at least one channel category assigned to it. When that condition is met, a checkmark will appear for the SKU under "Category Set" on the Available Channel SKUs page.

The publisher will assign the categories in most cases. However, as channel administrator, you can override the publisher's choices, or assign the SKUs to additional categories, if desired.

You will receive e-mail notification of any changes a publisher makes to channel categories after the initial setup. This e-mail will provide a link that you can use to directly log in to your channel account to change the SKU's channel categories. (More information on that process follows.)

To change a SKU's channel categories:

  1. From the Available Channel SKUs page, select the SKU.
  2. Click Edit Category. The Select SKU Channel Categories page appears.
  3. Follow the instructions provided to change the categories.

Note: There is no automated e-mail sent to the publisher when you change channel categories.

In Channel, Buy Qualified, Try Qualified – You can use the In Channel, Buy Qualified and Try Qualified settings to categorize SKUs for list purposes. These settings have no other effect on the SKUs.

Suggested uses for these settings are as follows:

  • To mark a SKU as "In Channel" after it has been added to your channel web site, select the SKU and then click Add to Channel.
  • To mark a SKU as "Buy Qualified" after its "Buy" file (or other purchasable option) has been tested and qualified for distribution in your channel, select the SKU and then click Mark as Buy Qualified.
  • To mark a SKU as "Try Qualified" after its "Try" file has been tested and qualified for distribution in your channel, select the SKU and then click Mark as Try Qualified.

Accessing SKU Settings

To access a SKU's settings, click its SKU field.
