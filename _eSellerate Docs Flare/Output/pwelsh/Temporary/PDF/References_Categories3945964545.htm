Creating Categories

Categories allow you to organize how your SKUs appear in search lists. You can also use categories to organize the display of SKUs in your eSellerate web stores. For information about assigning categories, see:

  • "Creating the SKU and Setting Its Basic Properties" on page 1
  • "Adding Embedded Web Store SKUs" on page 1
  • "Adding Web Store SKUs" on page 1
To create a category:
  1. Click the Categories tab. The Categories page appears.
  2. Click New. The Category Detail page appears.

  3. In the Category field, enter a name for this category.

    Note: If you use this category in your web stores, the name that you enter here will be visible to your customers.

  4. This category can apply to SKUs, web stores or both of these options. Select the appropriates check boxes.
  5. Click Save Changes.
