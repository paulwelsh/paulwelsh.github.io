CheckForUpdate Redirects

This report shows all the Integrated eSeller check for update redirect relationships set up for your eSellerate account.

To run a CheckForUpdate Redirects report:
  1. Click the Admin Reports tab. The Admin Reports page appears.
  2. Click CheckForUpdate Redirects. The resulting page displays redirect information, as in the example below.
