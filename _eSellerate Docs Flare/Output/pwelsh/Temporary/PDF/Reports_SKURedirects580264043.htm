SKU Redirects

This report shows all the SKU redirects set up for your eSellerate account.

To run a SKU Redirects report:
  1. Click the Admin Reports tab. The Admin Reports page appears.
  2. Click SKU Redirects. The resulting page displays redirect information, as in the example below.

For information on setting SKU redirect behavior, see "Adding Embedded Web Store SKUs" on page 1 and "Adding Web Store SKUs" on page 1.
