RPN

With this option, eSellerate generates serial numbers by using an RPN (Reverse Polish Notation) algorithm that you specify.

To create an RPN serial number set:
  1. From the Ready-to-Use Type list, select RPN.
  2. In the RPN Expression field, enter a legal RPN expression.

  3. You can use prompts to base your serial number generation on customer input. If you enable this feature, your eSellerate web stores (regular or embedded) will prompt customers to enter specific information as part of the purchase process.

    From the Prompt For list, select the information your web stores should require for SKUs that use this serial number set. Your choices are as follows:

    • None
    • RegCode
    • Palm OS® HotSync Name
    • Pocket PC Owner Name
    • (Custom Prompt)

    Note: When appropriate, online help will be available to show your customers how to locate handheld device information such as RegCode.

  4. The Custom Prompt text field appears/applies only when you select the "(Custom Prompt)" option in the Prompt For list. Text that you enter here (up to 50 characters) will define the prompt displayed to your customers. For example, if you enter "Nickname" in the Custom Prompt field, your web store will display "Nickname: [input field]" when that information is required for a SKU purchase.
  5. Click Save Changes.
