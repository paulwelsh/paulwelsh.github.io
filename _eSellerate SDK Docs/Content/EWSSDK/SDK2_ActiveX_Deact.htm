eWeb_DeactivateSerialNumber

For eSellerate Product Activation purposes, removes the association between a serial number and the unique hardware identification of the computer in use.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at Installing eWebControl365.dll (the recommended location). It may be already placed here by another eSellerate-enabled application.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

Integer eWeb_DeactivateSerialNumber ( publisherID, activationID, serialNumber, askToConnect )

publisherID – Publisher ID string

activationID – Product Activation Set ID string

serialNumber – serial number string

askToConnect – ask to connect when no active connection is found


See eSellerate Return Values for details on return values.

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a TRUE value to enable display of the dialog; otherwise pass FALSE.
Parameters for eWeb_DeactivateSerialNumber Function

Example

Dim esellerate As eWebSDKLib.eWebSDK

Dim activationResult As eWebSDKLib.eWeb_ReturnValues

Dim serialNumber as string = "xxxxx" ' fill in your serial number here

 

esellerate = CreateObject("eWebSDK.365")

 

activationResult = esellerate.eWeb_DeactivateSerialNumber("PUB775527079", "ACT123456789:QTNCLL-AVKJ-A73C62-EXPN-ECEBTK-89U5R0-VN2D-FCDW7J-UVTA-B1PHCA", serialNumber, false)
