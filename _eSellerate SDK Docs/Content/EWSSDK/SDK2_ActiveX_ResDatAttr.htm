eWebResultData Attributes

The following attributes can be retrieved directly from the eWebResultData class after creation of the eWebResultData object:

  • BillingFirstName
  • BillingLastName
  • BillingAddress1
  • BillingAddress2
  • BillingCity
  • BillingState
  • BillingZip
  • BillingCountry
  • BillingEmail
  • BillingPhone
  • BillingCompany
  • OrderID
  • OrderLineCount

Both lines that follow the variable declaration in the example code below retrieve the shopper's billing first name. The first line retrieves the data directly, and the second uses eWeb_GetResultDataValue.

Dim myBillingFirstName As string

 

myBillingFirstName = resultData.BillingFirstName

resultData.eWeb_IndexGetResultDataValue("_Shopper", "BillingFirstName", 0, myBillingFirstName)
