eWeb_ValidateSerialNumber

Checks the validity of a Standard or Standard Enhanced serial number.

Required Files

  • eWebControl365.dll
    Registered and typically located in the eSellerate directory noted at Installing eWebControl365.dll (the recommended location). It may be already placed here by another eSellerate-enabled application.

Definition

eSellerate_DaysSince2000 eWeb_ValidateSerialNumber ( serialNumber, nameBasedKey, extraDataKey, publisherKey )

serialNumber – serial number string

nameBasedKey – Name-based Key string

extraDataKey – additional validation key string

publisherKey – Publisher Key string

Tip: This function returns 0 if serialNumber is invalid. For a valid serialNumber, this function returns n, where n is a date expressed as the number of days since Jan. 1, 2000. If Duration is set on the Sales Manager Serial Number Set Detail page, then n is the date of expiration; otherwise, it is the date of purchase.

Parameter

Description

serialNumber

String containing the serial number. Required.

nameBasedKey

String containing the Name-based Key to be used to validate serialNumber. If the Based On setting on the Sales Manager Serial Number Set Detail page is set as None, pass "".

extraDataKey

String containing additional information to be used to validate serialNumber. If your Embedded Web Store eSeller did not use a special parameter (extraData) to pass additional information to the eSellerate servers during an API call to eWeb_Purchase or eWeb_CheckForUpdate, pass "".

publisherKey

String containing the Publisher Key to be used to validate serialNumber, as found on the Sales Manager Serial Number Set Detail page. Required.
Parameters for eWeb_ValidateSerialNumber Function

Caution: The publisherKey is not the same as publisherID used in functions that invoke the Embedded Web Store.

Note: For the key portion of a Standard or Standard Enhanced serial number (see "serial numbers: Serial Number Type options" in the eSellerate User's Guide index), eSellerate does not use the letters I, O, U or Z because they can be easily confused with other characters. To accommodate customers who mistakenly enter these letters in Standard or Standard Enhanced serial numbers, eWeb_ValidateSerialNumber will substitute the number 1 for I, the number 0 for O, the letter V for U and the number 2 for Z. The function also will substitute uppercase characters for lowercase ones.

Example

Dim esellerate As eWebSDKLib.eWebSDK

Dim result As eWebSDKLib.eWeb_ReturnValues

Dim serialNumber as string = "xxxxx" ' fill in your serial number here

 

esellerate = CreateObject("eWebSDK.365")

 

result = esellerate.eWeb_ValidateSerialNumber(serialNumber, "", "", "HST8-T063-05DY-H9F9-FNY5")

if (result > 0) then

' serial number is valid

Else

' serial number is invalid

End If

esellerate = Nothing
