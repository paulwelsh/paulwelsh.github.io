eWeb_GetResultDataValueLength

Retrieves the length (in bytes) of a result value given its name. It is a utility function for properly allocating memory in which to store result data. This API should be called before reading data value to prevent buffer overflows.

Tip: This API can retrieve the length of result values for indexed and non-indexed eSellerate variables. Indexed variables are designated as "Loop Only" on the variables page at http://publishers.esellerate.net/embeddedvar.aspx.

Required Files

  • EWSEmbedded.framework (standard) or
    EWSEmbedded-GC.framework (garbage collecting)
    Included in the development project.

Definition

OSStatus eWeb_GetResultDataValueLength ( resultData, Name, Length )

resultData – results obtained from the Purchase or CheckForUpdate family of functions

Name – name of the result data variable for which to search

Length – length of the variable's value


For a listing of all eSellerate variables that Embedded Web Store eSellers can pull from result data, use the following URL: http://publishers.esellerate.net/embeddedvar.aspx.

See eSellerate Return Values for details on return values.

Parameter

Description

resultData

Address of a declared variable of type char *. This should have already been filled by eWeb_Purchase or eWeb_CheckForUpdate.

Name

The name of the result data variable for which to search. Required. Must include a zero-based index if the variable is designated as "Loop Only" on the variables page at
http://publishers.esellerate.net/embeddedvar.aspx. For example, use "_OrderItem0.SerialNumber" to search for the serial number issued for the first order item.

Length

The length of the variable's value. Required.
Parameters for eWeb_GetResultDataValueLength Function

Note: All strings passed into the engine are UTF-8 encoded cStrings.

Example

#import<EWSEmbedded/EWSEmbedded.h>

char *resultData = NULL;

WORD valueLength;

/* TO DO: obtain resultData by calling an Embedded Web Store invoking function (Purchase, CheckForUpdate) */

OSStatus resultCode = eWeb_GetResultDataValueLength(resultData, "_OrderItem0.SerialNumber", &valueLength);

/* TO DO: create a buffer to store the serial number using valueLength to determine the size of the buffer */

eWeb_DisposeResultData(resultData);
