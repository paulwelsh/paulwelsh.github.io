eWeb_IndexGetResultDataValue

Returns the data for the supplied record/field combination. In order to provide enough memory for the resultant data value, call eWeb_IndexGetResultDataLength.

Tip: This API can retrieve the value of indexed eSellerate variables only. Indexed variables are designated as "Loop Only" on the variables page at http://publishers.esellerate.net/embeddedvar.aspx. To retrieve the value of a non-indexed variable, use Web_GetResultDataValue.

  • EWSLib.h
    Included in the development project.
  • libEWS.a (standard) or
    libEWS-GC.a (garbage collecting)
    Included in the development project.

    Included in the development project.

Definition

OSStatus eWeb_IndexGetResultDataValue ( resultData, Record, Field, Index, Value, ValueLen )

resultData – results obtained from the Purchase or CheckForUpdate family of functions

Record – name of the record containing the variable being searched

Field – name of the field within the record for which you wish to search

Index – zero-based index of the record/field instance

Value – buffer to receive the value upon success

ValueLen – length of the buffer to receive the value upon success


For a listing of all eSellerate variables that Embedded Web Store eSellers can pull from result data, use the following URL: http://publishers.esellerate.net/embeddedvar.aspx.

See eSellerate Return Values for details on return values.

Parameter

Description

resultData

Address of a declared variable of type char *. This should have already been filled by eWeb_Purchase or eWeb_CheckForUpdate.

Record

Name of the record containing the variable being searched. Required. An example of a valid record name follows:

Complete variable name: _CartItem.Quantity
Record name: _CartItem

Field

Name of the field within the record for which you wish to search. Required. An example of a valid field name follows:

Complete variable name: _CartItem.Quantity
Field name: Quantity

Index

Zero-based index of the record/field instance. Required.

Value

Buffer to receive the value upon success. Required.

ValueLen

Length of the buffer to receive the value upon success. Required.
Parameters for eWeb_IndexGetResultDataValue Function

Note: All strings passed into the engine are UTF-8 encoded cStrings.
