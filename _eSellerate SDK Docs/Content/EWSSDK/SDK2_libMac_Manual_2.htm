eWeb_ManualActivateSerialNumber

Launches the Embedded Web Store Engine to allow clients to manually activate a serial number. Typically this API is used when the current machine cannot get a network connection and a serial number needs to be activated.

Also see "Product Activation: manually activating serial numbers" in the eSellerate User's GuidePublishers Help index.

Required Files

  • EWSEmbedded.framework (standard) or
    EWSEmbedded-GC.framework (garbage collecting)
    Included in the development project.

Definition

OSStatus eWeb_ManualActivateSerialNumber ( publisherID, activationID, serialNumber, titlePreface, otherOptions )

publisherID – Publisher ID string

activationID – Product Activation Set ID string

serialNumber – serial number string

titlePreface – custom text, to preface activation wizard titles (optional)

otherOptions – custom text, to include phone number, e-mail or other contact (optional)


See eSellerate Return Values for details on return values.

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

titlePreface

String containing a custom preface to the titles of all Manual Product Activation Wizard pages. Causes the titles to be "<Preface>: Product Activation." May be NULL, in which case the titles will simply be "Product Activation."

otherOptions

String containing custom information to help users without web access activate their software. (This might be contact information for the publisher, who would visit http://activate.esellerate.net to obtain an Activation Key for the user.) The wizard will display up to six lines for this string. May be NULL, in which case the option to activate without web access will not be available to users.
Parameters for eWeb_ManualActivateSerialNumber Function

Note: All strings passed into the engine are UTF-8 encoded cStrings.

Note: Unless there is a specific need for your application source code to directly invoke the Manual Product Activation Wizard, it is more efficient to use the eWeb_ActivateSerialNumber function instead of eWeb_ManualActivateSerialNumber.

When used with the askToConnect parameter set to TRUE, eWeb_ActivateSerialNumber can handle normal (automatic) activations and also invoke the Manual Product Activation Wizard when needed. In contrast, eWeb_ManualActivateSerialNumber can only handle manual activations, and consequently is limited to use in special circumstances.

Example

#import<EWSEmbedded/EWSEmbedded.h>

char *serialNumber;

OSStatus resultCode;

/* TO DO: obtain serialNumber */

resultCode = eWeb_ManualActivateSerialNumber("PUB775527079", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU", serialNumber, NULL, NULL);

/* TO DO: handle error, or react to success */
