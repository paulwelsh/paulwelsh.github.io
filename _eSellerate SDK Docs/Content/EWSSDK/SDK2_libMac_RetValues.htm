eSellerate Return Values

The following are eSellerate-specific return values and their descriptions.

Tip: Numeric equivalents for the following return values are available in the latest header file (EWSLib.h). In general, negative return values represent errors, positive values represent non-errors, and zero represents success.

Return

Description

E_SUCCESS

The call completed successfully.

E_SDK_INSTALLED_ENGINE

The latest engine was successfully installed.

E_SDK_LATEST_ENGINE_ALREADY_INSTALLED

The latest engine is already installed.

E_SDK_CREATE_ENTRY_ERROR

The SDK failed to add a name/value entry.

E_SDK_BAD_PARAMETER

A parameter is incorrect.

E_SDK_ENGINE_NOT_INSTALLED

There was an error installing the engine.

E_SDK_ENGINE_NOT_LOADED

There was an error loading the installed engine.

E_SDK_ENGINE_CORRUPTED

The engine is corrupted.

E_SDK_OBJECT_NOT_FOUND

The result from the engine could not be found.

E_SDK_ENGINE_BUSY

The engine is already running within the application.

E_SDK_INSTALL_CORRUPTED

The engine being installed is corrupted.

E_INET_CONNECTION_FAILURE

A general connection failure has occurred.

E_INET_ESELLERATE_FAILURE

The engine failed to connect to the eSellerate servers.

E_INET_DOWNLOAD_ENGINE_FAILURE

The engine failed to download an update.

E_INET_ESTATE_NOT_FOUND

The eSellerate Engine data is missing from the results.

E_VALIDATEACTIVATION_MACHINE_MATCH

The current machine matches the machine on which the product was activated.

E_VALIDATEACTIVATION_MACHINE_MISMATCH

The current machine does not match the machine on which the product was activated.

E_VALIDATEACTIVATION_OLD_ACTIVATION_KEY

The activation key is an older format that is invalid.

E_ACTIVATION_MANUAL_CANCEL

The user canceled a manual activation attempt.

E_ACTIVATION_REQUEST_DENIED

eSellerate denied the activation request.

E_ACTIVATESN_UNKNOWN_ACTIVATION_KEY

The format of the supplied activation key is invalid.

E_ACTIVATESN_UNKNOWN_SN

An attempt was made to activate an unknown serial number.

E_ACTIVATESN_IMPROPER_USAGE

Improper usage of the activation routines.

E_ACTIVATESN_BLACKLISTED_SN

An attempt was made to activate a blacklisted serial number.

E_ACTIVATESN_INVALID_ORDER

An activation attempt was made on an invalid order.

E_ACTIVATESN_LIMIT_MET

An activation attempt failed due to the maximum number of allowable activations being met.

E_ACTIVATESN_NOT_UNIQUE

The serial number to be activated is not unique.

E_ACTIVATESN_UNKNOWN_SERVER_ERROR

An unknown server error has occurred.

E_ACTIVATESN_FINALIZATION_ERROR

An error occurred finalizing the activation on the client machine.

E_ACTIVATESN_NO_SN_FROM_SERVER

The eSellerate servers failed to return a valid activation key.

E_CONFIRMATION_NO_SUCH_SERIAL_NUMBER

An attempt was made to confirm an unknown serial number.

E_CONFIRMATION_PUBLISHER_SN_MISMATCH

A confirmation attempt failed because the supplied serial number does not correspond to the supplied publisher ID.

E_CONFIRMATION_BLACKLISTED_SERIAL_NUMBER

An attempt was made to confirm a blacklisted serial number.

E_CONFIRMATION_INVALID_ORDER

A confirmation attempt was made on an invalid order.

E_CONFIRMATION_UNEXPECTED_FAILURE

A confirmation attempt failed due to an unexpected error.

E_DEACTIVATION_NO_SUCH_SERIAL_NUMBER

An attempt was made to deactivate an unknown serial number.

E_DEACTIVATION_BLACKLISTED_SERIAL_NUMBER

An attempt was made to deactivate a blacklisted serial number

E_DEACTIVATION_INVALID_ORDER

A deactivation attempt was made on an invalid order.

E_DEACTIVATION_MISSING_ACTIVATION_KEY

A deactivation attempt failed because the supplied activation key is unknown.

E_DEACTIVATION_MACHINE_MISMATCH

A deactivation attempt failed because the current machine does not match the machine on which the product was activated.

E_DEACTIVATION_INVALID_ACTIVATION_KEY

A deactivation attempt failed because the supplied activation key is invalid.

E_DEACTIVATION_NONUNIQUE_SERIAL_NUMBER

A deactivation attempt failed because the serial number to be deactivated is not unique.

E_DEACTIVATION_NO_SUCH_ACTIVATION_KEY

A deactivation attempt failed because the supplied activation key is unknown.

E_DEACTIVATION_DEACTIVATION_LIMIT_MET

A deactivation attempt failed due to the maximum number of allowable deactivations being met.

E_ENGINE_SKU_UPDATE_AVAILABLE

A SKU update is available from the eSellerate servers.

E_ENGINE_SKU_NO_UPDATE_AVAILABLE

No SKU update is available from the eSellerate servers.

E_ENGINE_INTERNAL_ERROR

An internal error has occurred in the engine.

E_ENGINE_PURCHASE_NOTSUCCESSFUL

A purchase attempt was unsuccessful.

E_OS_FUNCTIONALITY_UNSUPPORTED

The system does not meet the minimum requirements for the Embedded Web Store.
