Default Locations for eWebClient.dll

The default location for the eSellerate Embedded Web Store Engine (eWebClient.dll) on the user's machine varies according to the Windows operating system in use.

For Windows NT-based operating systems including Windows NT, Windows 2000 and Windows XP, the default location is the Program Files\Common Files\eSellerate directory. A typical path is C:\Program Files\Common Files\eSellerate\eWebClient.dll.

For Windows Vista, the default location is the ProgramData\eSellerate directory. A typical path is C:\ProgramData\eSellerate\eWebClient.dll.

Note: On 64-bit Windows editions such as Windows XP Professional x64 Edition, the default location for the engine will include "(x86)" in its path. An example is C:\Program Files (x86)\Common Files\eSellerate\eWebClient.dll.

For information on APIs that you can use to install the engine, see eWeb_InstallEngineFromResource and eWeb_InstallEngineFromPath.
