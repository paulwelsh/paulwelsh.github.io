eWeb_GetResultDataValueLength

Retrieves the length (in bytes) of a result value given its name. It is a utility function for properly allocating memory in which to store result data. This API should be called before reading data value to prevent buffer overflows.

Tip: This API can retrieve the length of result values for indexed and non-indexed eSellerate variables. Indexed variables are designated as "Loop Only" on the variables page at http://publishers.esellerate.net/embeddedvar.aspx.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

HRESULT eWeb_GetResultDataValueLength ( resultData, Name, Length )

resultData – results obtained from the Purchase or CheckForUpdate family of functions

Name – name of the result data variable for which to search

Length – length of the variable's value


For a listing of all eSellerate variables that Embedded Web Store eSellers can prefill or pull from result data, use the following URL: http://publishers.esellerate.net/embeddedvar.aspx.

See eSellerate Return Values for details on return values.

Parameter

Description

resultData

Address of a declared variable of type BYTE *. This should have already been filled by eWeb_Purchase, eWeb_PurchaseEx, eWeb_CheckForUpdate or eWeb_CheckForUpdateEx.

Name

The name of the result data variable for which to search. Required. Must include a zero-based index if the variable is designated as "Loop Only" on the variables page at http://publishers.esellerate.net/embeddedvar.aspx. For example, use "_OrderItem0.SerialNumber" to search for the serial number issued for the first order item.

Length

The length of the variable's value. Required.
