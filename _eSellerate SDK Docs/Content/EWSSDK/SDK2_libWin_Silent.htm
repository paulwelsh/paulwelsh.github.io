eWeb_SilentCheckForUpdate

Contacts the eSellerate servers without invoking the Embedded Web Store Engine in order to determine if an update is available for the identified SKU.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

HRESULT eWeb_SilentCheckForUpdate ( esellerID, SkuRefNum, previewID, AskToConnect, trackingID )

esellerID – eSeller RefNum string

SkuRefNum – SKU RefNum

previewID – Preview Certificate ID code (optional)

askToConnect – ask to connect when no active connection is found

trackingID – reserved for future functionality


See eSellerate Return Values for details on return values.

Parameter

Description

esellerID

String containing the Embedded Web Store ID, as found on the Sales Manager Embedded Web Store eSeller Summary page. Required.

SkuRefNum

String containing the SKU RefNum, as found on the Sales Manager Embedded Web Store SKU Attributes page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Embedded Web Store eSeller Summary page. Required to purchase in Preview mode. Pass NULL in Active mode. If the Preview Certificate passed here does not match the Preview Certificate on the Sales Manager Embedded Web Store eSeller Summary page, the purchase will be Active. This means that you can clear the Preview Certificate in the Sales Manager to activate the eSeller without having to recompile your application.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a TRUE value to enable display of the dialog; otherwise pass FALSE.

trackingID

String reserved for future functionality. Pass NULL.
