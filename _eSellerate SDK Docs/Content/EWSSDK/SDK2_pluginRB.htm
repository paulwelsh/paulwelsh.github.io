eWeb Plug-in for REALbasic

This section details the eSellerate Embedded Web Store SDK provided by the REALbasic plug-in option for eSeller integration into Macintosh and Windows applications developed using REALbasic. It documents the methods of the eStore class:

  • IsSystemCompatible
  • InstallEngine
  • Purchase
  • CheckForUpdate
  • SilentCheckForUpdate
  • SetOption
  • GetOrderCount
  • GetOrderItemAttribute
  • ActivateSerialNumber
  • ManualActivateSerialNumber
  • ValidateActivation
  • DateActivation
  • DeactivateSerialNumber
  • ConfirmSerialNumber
  • ValidateSerialNumber
  • StandardizeSerialNumber
  • Today
  • TranslateError
  • GetLastError
Notes
  • The system requirements for this option are as follows:
    • Macintosh — Mac OS X v10.3.9 or later
    • Windows — Windows XP or later, or Windows 2000 or later, and Internet Explorer 5.5 or later
  • For a summary of previous and current function calls applicable to the eWeb plug-in for REALbasic, see New Functions Summary.
  • This note applies to eSeller integration into Macintosh applications only. EWS for REALbasic is a Universal (universal binary) solution that runs natively on both PowerPC- and Intel-based Macintosh computers. (The system requirements are listed earlier in these notes.)
  • This note applies to eSeller integration into Windows applications only. EWS for REALbasic is compatible with 64-bit Windows editions such as Windows XP Professional x64 Edition. This is not a 64-bit solution, but you can integrate it with your 32-bit applications to allow Embedded Web Store functionality on 64-bit Windows.
  • In contrast to other EWS options, EWS for REALbasic does not support prefilling or reading eSellerate variables that are used in transactions.
