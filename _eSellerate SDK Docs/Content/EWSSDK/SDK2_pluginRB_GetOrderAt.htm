GetOrderItemAttribute

Extracts information about a single order item from the results obtained by calling an Embedded Web Store invoking function. (To determine the total number of order items, use GetOrderCount.)

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.
Macintosh targets
  • EWSMac.framework
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.
Windows targets
  • eWebClient.dll
    Located in the eSellerate directory noted at Default locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

GetOrderItemAttribute ( itemIndex, attributeName ) as String

itemIndex as Integer – zero-based index of the order item for which you are searching

attributeName as String – name of the item attribute you want to retrieve


See eSellerate Return Values for details on return values.

Parameter

Description

itemIndex

Zero-based index of the order item for which you are searching. Required.

attributeName

Name of the item attribute you want to retrieve. Use one of the defined attribute names listed in the next table. Required.
Parameters for GetOrderItemAttribute Function

Attribute Name

Description

"SkuRefNum"

SKU RefNum for the original SKU purchased.

"Quantity"

Quantity of the item purchased.

"SerialNumber"

Serial number associated with the order item.

"SerialNumberKey"

Name on which the serial number was based.

"PromptedValue"

Prompted value entered by the user for use in serial number generation.

"DownloadURL"

eSellerate URL from which this order item may be downloaded.

Attributes for GetOrderItemAttribute Function
