eSellerate Return Values

In general, negative return values represent errors, positive values represent non-errors, and zero represents success.

eSellerate-specific return values are listed here as follows:

  • "Return Value" is the value the eWeb plug-in returns.
  • "Translation" is the string that you can retrieve by passing the return value in the errorCode parameter of the TranslateError function.
  • "Description" provides more information about the return value.

Return Value

Translation

Description

0

SUCCESS

The call completed successfully.

-1

FAILURE

The call was unsuccessful.

-2

NO ENGINE

No engine is installed.

-3

USER CANCELED

The user canceled a manual activation attempt.

-4

NO UPDATE

No SKU update is available from the eSellerate servers.

-25000

UNKNOWN ACTIVATION ID

An attempt to activate or deactivate a serial number failed because the supplied activation ID is unknown.

-25001

UNKNOWN SERIAL NUMBER

An attempt to activate, deactivate or confirm a serial number failed because the supplied serial number is unknown.

-25002

MISSING SERIAL NUMBER

An attempt to activate, deactivate or confirm a serial number failed because no serial number was passed in the call.

-25003

BLACKLISTED SERIAL NUMBER

An attempt to activate, deactivate or confirm a serial number failed because the supplied serial number is blacklisted.

-25004

INVALID ORDER

An attempt to activate, deactivate or confirm a serial number failed because the associated order is invalid. Typically, an order is invalid because it was returned.

-25005

ACTIVATION LIMIT MET

An activation attempt failed due to the maximum number of allowable activations being met.

-25006

UNKNOWN ACTIVATION KEY

A deactivation attempt failed because the supplied activation key is unknown.

-25007

ACTIVATION KEY/HARDWARE MISMATCH

An attempt to validate activation or to deactivate a serial number failed because the current machine does not match the machine on which the product was activated.

-25008

INVALID ACTIVATION KEY

A deactivation attempt failed because the supplied activation key is invalid.

-25009

NON-UNIQUE SERIAL NUMBER

An attempt to activate or deactivate a serial number failed because the supplied serial number is not unique.

-25010

NO SUCH ACTIVATION KEY

A deactivation attempt failed because the supplied activation key is unknown.

-25011

DEACTIVATION LIMIT MET

A deactivation attempt failed due to the maximum number of allowable deactivations being met.

-25012

REQUEST DENIED

eSellerate denied the activation request.

All other error codes

UNDEFINED

The error code is not one of those defined for the eWeb plug-in for REALbasic. Therefore, no further information is available by calling TranslateError.
