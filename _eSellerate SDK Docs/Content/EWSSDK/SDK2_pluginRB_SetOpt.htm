SetOption

Assigns a value to options available to the following functions: Purchase, CheckForUpdate, SilentCheckForUpdate, ActivateSerialNumber, ManualActivateSerialNumber, DeactivateSerialNumber and ConfirmSerialNumber.

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.
Macintosh targets
  • EWSMac.framework
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.
Windows targets
  • eWebClient.dll
    Located in the eSellerate directory noted at Default locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

SetOption ( optionName, optionValue )

optionName as String – defined name of option

optionValue as String – value to be defined to option

Parameter

Description

optionName

String containing one of the defined option names listed in the next table. Required.

optionValue

String containing a value to be assigned to the option. Required.

Option Details for SetOption Function

Note: Option settings are "" (or "FALSE" when applicable) until set by SetOption.

Functions use the latest setting set by SetOption.

Call SetOption before calling the function(s) that will use it.

There is no need to reset an option setting between function calls, unless that setting needs to change.

Example

dim eSellerate as eStore

dim resultCode as Integer

eSellerate = new eStore

eSellerate.SetOption("SkuRefNum", "SKU943434078")

eSellerate.SetOption("ActivationID", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU")

resultCode = eSellerate.Purchase("STR9199856688")
