ValidateSerialNumber

Checks the validity of a Standard or Standard Enhanced serial number.

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.

Definition

ValidateSerialNumber ( serialNumber, nameBasedKey, extraDataKey, publisherKey ) as Integer

serialNumber as String – serial number string

nameBasedKey as String – Name-based Key string

extraDataKey as String – additional validation key string

publisherKey as String – Publisher Key string

Tip: This function returns 0 if serialNumber is invalid. For a valid serialNumber, this function returns n, where n is a date expressed as the number of days since Jan. 1, 2000. If Duration is set on the Sales Manager Serial Number Set Detail page, then n is the date of expiration; otherwise, it is the date of purchase.

Parameter

Description

serialNumber

String containing the serial number. Required.

nameBasedKey

String containing the Name-based Key to be used to validate serialNumber. If the Based On setting on the Sales Manager Serial Number Set Detail page is set as None, pass "".

extraDataKey

String containing additional information to be used to validate serialNumber. If your Embedded Web Store eSeller did not use a special parameter (extraData) to pass additional information to the eSellerate servers during an API call to Purchase or CheckForUpdate, pass "".

publisherKey

String containing the Publisher Key to be used to validate serialNumber, as found on the Sales Manager Serial Number Set Detail page. Required.
Parameters for ValidateSerialNumber Function

Caution: The publisherKey is not the same as publisherID used in functions that invoke the Embedded Web Store.

Note: For the key portion of a Standard or Standard Enhanced serial number (see "serial numbers: Serial Number Type options" in the eSellerate User's Guide index), eSellerate does not use the letters I, O, U or Z because they can be easily confused with other characters. To accommodate customers who mistakenly enter these letters in Standard or Standard Enhanced serial numbers, ValidateSerialNumber will substitute the number 1 for I, the number 0 for O, the letter V for U and the number 2 for Z. The function also will substitute uppercase characters for lowercase ones.
