GetFileLocation

Retrieves the location of a file downloaded by a Wizard-invoking function from the information organized by GetFileLocationCount.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Definition

GetFileLocation (

index As Long – file number

) As String=

Return Value

Description

""

Returned if there is no corresponding downloaded file.

file location

Returned if there is a corresponding downloaded file, where file location is the location of that file.
Return Values for GetFileLocation Function

Parameter

Description

index

File number. Indexing runs from 1 up to the count returned by GetFileLocationCount. Required.
Parameters for GetFileLocation Function

Note: File locations are expressed as pathnames.

Note: GetFileLocationCount must be called before GetFileLocation is called.

Example

Dim eSellerate As eSeller

Dim fileLocation As String

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: call a Wizard-invoking function **

If eSellerate.GetFileLocationCount > 0 Then

fileLocation = eSellerate.GetFileLocation(1)

' ** TO DO: utilize fileLocation **

End If

Set eSellerate = Nothing
