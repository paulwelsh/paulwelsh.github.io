GetSerialNumberKey

Retrieves a Name-based Key from the serial number information obtained by a Wizard-invoking function and organized by GetSerialNumberCount.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Definition

GetSerialNumberKey (

index As Long – item number

) As String

Return Value

Description

""

Returned if no Name-based Key associated with the serial number for the item specified is available, possibly because there is no serial number for the item or because there is no key associated with the serial number.

serial number key

Returned if a Name-based Key associated with the serial number for the item specified is available, where that Name-based Key is the string serial number key.
Return Values for GetSerialNumberKey Function

Parameter

Description

index

Item number. Indexing runs from 1 up to the count returned by GetSerialNumberCount. Required.
Parameters for GetSerialNumberKey Function

Note: GetSerialNumberCount must be called before GetSerialNumberKey is called.

Note: The Name-based Key for a serial number is always "" if the Based On setting on the Sales Manager Serial Number Set Detail page is set as None.

Example

Dim eSellerate As eSeller

Dim nameBasedKey As String

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: call a Wizard-invoking function **

If eSellerate.GetSerialNumberCount > 0 Then

nameBasedKey = eSellerate.GetSerialNumberKey(1)

' ** TO DO: utilize nameBasedKey **

End If

Set eSellerate = Nothing
