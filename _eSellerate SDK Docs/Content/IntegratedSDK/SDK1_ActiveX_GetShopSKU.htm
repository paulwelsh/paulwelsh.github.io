GetShoppingCartSKU

Retrieves the SKU ID of an item obtained by a Wizard-invoking function from the information organized by GetShoppingCartCount.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Definition

GetShoppingCartSKU (

index As Long – item number

) As String

Return Value

Description

""

Returned if there is no corresponding shopping cart item.

shopping cart SKU

Returned if there is a corresponding shopping cart item, where shopping cart SKU is the SKU ID of that item.
Return Values for GetShoppingCartSKU Function

Parameter

Description

index

Item number. Indexing runs from 1 up to the count returned by GetShoppingCartCount. Required.
Parameters for GetShoppingCartSKU Function

Note: GetShoppingCartCount must be called before GetShoppingCartSKU is called.

Example

Dim eSellerate As eSeller

Dim skuIDArray(64) As String

Dim count, index As Long

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: call a Wizard-invoking function **

count = eSellerate.GetShoppingCartCount

For index = 1 To count

skuIDArray(index) = eSellerate.GetShoppingCartSKU(index)

Next index

' ** TO DO: utilize skuIDArray **

Set eSellerate = Nothing
