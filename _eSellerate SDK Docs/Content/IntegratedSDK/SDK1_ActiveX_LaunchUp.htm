LaunchUpdate

Launches a standalone executable, such as an installer, or applies an Updater VISE file to a specified file and launches the resulting updated file.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

LaunchUpdate (

updaterFile As String, – location of an Updater VISE file to apply or an executable to launch

fileToUpdate As String – location of the file that an Updater VISE file is to be applied to

) As Long

Return Value

Description

-2

Returned if eSellerateEngine cannot be found.

-1

Returned if updaterFile cannot be launched.

0

Returned if updaterFile does launch.
Return Values for LaunchUpdate Function

Parameter

Description

updaterFile

Location of either an Updater VISE file to apply to fileToUpdate or a standalone executable to launch. Required.

fileToUpdate

Location of the file to apply updaterFile to, when updaterFile is an Updater VISE file. If it is not, pass "" to launch a standalone executable, such as an installer.
Parameters for LaunchUpdate Function

Note: File locations are expressed as pathnames.

Note: It may well be the case that a running application calls LaunchUpdate for the purpose of updating itself. In this situation, the application should promptly terminate itself after calling LaunchUpdate, since the application is then to be replaced.

Example

Dim eSellerate As eSeller

Dim updaterFile As String

Dim fileToUpdate As String

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: obtain updaterFile pathname and, if needed, updaterFile pathname **

eSellerate.LaunchUpdate(updaterFile, fileToUpdate)

Set eSellerate = Nothing

' ** TO DO: if fileToUpdate is the running application, exit application **
