Reinstall

Invokes the Software Delivery Wizard to enable the user to reinstall software from a previous purchase.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

Reinstall (

publisherID As String, – Publisher ID string

esellerID As String, – Integrated eSeller RefNum string

previewID As String, – Preview Certificate string

errorURL As String, – URL string shown in error messages

openingTitle As String, – custom title of opening page of the Software Delivery Wizard

openingText As String, – custom first paragraph of opening page of the Software Delivery Wizard

showFolder As Long, – have the eSellerateEngine open a window to display downloaded files

trackingID As String, – tracking ID code

activationID As String – Product Activation ID Set string

) As Long

Return Value

Description

-3

Returned if the user exits the eSellerateEngine before the action was completed.

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if the eSellerateEngine encounters a fatal error during reinstall.

0

Returned if the eSellerateEngine completes the action.

-25000

Returned if the eSellerateEngine completes the action, but activation fails because activationID was unknown. (Only possible if activationID is present.)

-25002

Returned if the eSellerateEngine completes the action, but no activation occurs because there is no serial number associated with the SKU. (Only possible if activationID is present.)

-25003

Returned if the eSellerateEngine completes the action, but activation fails because the serial number is blacklisted. (Only possible if activationID is present.)

-25004

Returned if the eSellerateEngine completes the action, but activation fails because the product was returned. (Only possible if activationID is present.)

-25005

Returned if the eSellerateEngine completes the action, but activation fails because the activation limit was met. (Only possible if activationID is present.)

-25009

Returned if the eSellerateEngine completes the action, but activation fails because the serial number obtained by calling Reinstall is already associated with another order. (Only possible if activationID is present.)

-25012

Returned if eSellerate denies the activation request. (Only possible if activationID is present.)
Return Values for Reinstall Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

esellerID

String containing the Integrated eSeller RefNum, as found on the Sales Manager Integrated eSeller Summary page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Integrated eSeller Summary page. Required to purchase in Preview mode. Pass "" in Active mode.

errorURL

String containing a contact URL for the Software Delivery Wizard to use in the text of certain message boxes when reporting an error. May be "".

openingTitle

String containing a custom title of the opening page of the Software Delivery Wizard. May be "", in which case the system default title will be used.

openingText

String containing a custom first paragraph of the opening page of the Software Delivery Wizard. The limit is two lines, or approximately 150 characters, depending on character width. May be "", in which case the system default first paragraph will be used.

showFolder

Logical value that controls whether the eSellerateEngine opens a window after the purchase to display downloaded files. Pass a true (non-zero) value to open the folder; otherwise pass false (zero).

trackingID

String containing special information that marks the function call so that it can be tracked in reports. May be "". See "tracking IDs" in the eSellerate User's GuidePublishers Help index.

activationID

String containing an activation ID for use in eSellerate Product Activation. May be "". Automatic activation occurs if the eSellerateEngine completes the action and returns a unique serial number for use with the activationID (barring any -25000 series errors). See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

Dim eSellerate As eSeller

Dim resultCode As Long

Set eSellerate = CreateObject("eSellerateControl.365")

resultCode = eSellerate.Reinstall("PUB775527079", "ES123456789", "", "", "", "", 0, "", "")
