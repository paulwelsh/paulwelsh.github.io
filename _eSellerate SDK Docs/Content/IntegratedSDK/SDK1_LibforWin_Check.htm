eSellerate_CheckForUpdate

Invokes the Software Delivery Wizard to check for an update.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

eSellerate_ErrorCode eSellerate_CheckForUpdate ( publisherID, esellerID, previewID, errorURL, openingTitle, openingText, showFolder, trackingID, affiliateID, couponID, activationID, extraData, resultData )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString esellerID – Integrated eSeller RefNum string

eSellerate_InputString previewID – Preview Certificate string

eSellerate_InputString errorURL – URL string shown in error messages

eSellerate_InputString openingTitle – custom title of opening page of the Software Delivery Wizard

eSellerate_InputString openingText – custom first paragraph of opening page of the Software Delivery Wizard

eSellerate_TrueOrFalse showFolder – have the eSellerateEngine open a window to display downloaded files

eSellerate_InputString trackingID – tracking ID code

eSellerate_InputString affiliateID – affiliate ID code

eSellerate_InputString couponID – coupon ID code

eSellerate_InputString activationID – Product Activation Set ID string

eSellerate_InputString extraData – special serial number data string

eSellerate_ResultData* resultData – address of results obtained by the Software Delivery Wizard

Return Value

Description

eSellerate_NOENGINE

Returned if the eSellerateEngine cannot be found.

eSellerate_FAILURE

Returned if publisherID or esellerID is NULL or if the eSellerateEngine encounters a fatal error during check for update.

eSellerate_SUCCESS

Returned if the eSellerateEngine completes the action.

eSellerate_CANCELED

Returned if the user exits the eSellerateEngine before the action was completed.

-25000

Returned if the eSellerateEngine completes the action, but activation fails because activationID was unknown. (Only possible if activationID is present.)

-25009

Returned if the eSellerateEngine completes the action, but activation fails because the serial number obtained by calling eSellerate_CheckForUpdate is already associated with another order. (Only possible if activationID is present.)

-25012

Returned if eSellerate denies the activation request. (Only possible if activationID is present.)
Return Values for eSellerate_CheckForUpdate Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

esellerID

String containing the Integrated eSeller RefNum, as found on the Sales Manager Integrated eSeller Summary page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Integrated eSeller Summary page. Required to purchase in Preview mode. Pass NULL in Active mode.

errorURL

String containing a contact URL for the Software Delivery Wizard to use in the text of certain message boxes when reporting an error. May be NULL.

openingTitle

String containing a custom title of the opening page of the Software Delivery Wizard. May be NULL, in which case the system default title will be used.

openingText

String containing a custom first paragraph of the opening page of the Software Delivery Wizard. The limit is two lines, or approximately 150 characters, depending on character width. May be NULL, in which case the system default first paragraph will be used.

showFolder

Logical value that controls whether the eSellerateEngine opens a window after the purchase to display downloaded files. Pass a true (non-zero) value to open the folder; otherwise pass false (zero).

trackingID

String containing special information that marks the function call so that it can be tracked in reports. May be NULL. See "tracking IDs" in the eSellerate User's GuidePublishers Help index.

affiliateID

String containing an affiliate ID that marks the function call so that it can be tracked in reports. May be NULL. For information on the eSellerate Affiliates program, see "eSellerate Affiliates: joining" in the eSellerate User's GuidePublishers Help index.

couponID

String containing a coupon ID that enables coupon discounts on qualified purchases. May be NULL. See "coupons: creating" in the eSellerate User's GuidePublishers Help index.

activationID

String containing an activation ID for use in eSellerate Product Activation. May be NULL. Automatic activation occurs if the eSellerateEngine completes the action and returns a unique serial number for use with the activationID (barring any -25000 series errors). See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

extraData

String containing special additional information needed for serial number generation. Required for serial number type option Application Supplied and optional for Standard, Standard Enhanced and Armadillo. For all other serial number type options, pass NULL. See "serial numbers: Serial Number Type options" in the eSellerate User's GuidePublishers Help index.

resultData

Address of a declared variable of type ResultData. The variable is a pointer, and may be initialized to NULL. Memory is allocated and filled by the Software Delivery Wizard when eSellerate_CheckForUpdate is called. This memory must later be de-allocated by calling eSellerate_DeleteResultData.
