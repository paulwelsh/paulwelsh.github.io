eSellerate_ConfirmSerialNumber

Confirms that a serial number is valid within the eSellerate system. A serial number is invalid if it is unknown, blacklisted or associated with an invalid order.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

eSellerate_ErrorCode eSellerate_ConfirmSerialNumber ( publisherID, serialNumber, askToConnect )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString serialNumber – serial number string

eSellerate_TrueOrFalse askToConnect – ask to connect when no active connection is found

Return Value

Description

eSellerate_NOENGINE

Returned if the eSellerateEngine cannot be found.

eSellerate_FAILURE

Returned if publisherID or serialNumber is NULL or if the eSellerateEngine is unable to connect to the Internet. When no active connection to the Internet is found, the engine will return eSellerate_FAILURE if the value of the askToConnect parameter is false (zero) or if the value is true (non-zero) and the user cancels the dialog that asks for permission to connect.

eSellerate_SUCCESS

Returned if the eSellerateEngine completes the action.

-25001

Returned if confirmation fails because serialNumber was unknown.

-25003

Returned if confirmation fails because serialNumber is blacklisted.

-25004

Returned if confirmation fails because serialNumber is associated with an invalid order.
Return Values for eSellerate_ConfirmSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

serialNumber

String containing the serial number. Required.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a true (non-zero) value to enable display of the dialog; otherwise pass false (zero).

Type Definitions for eSellerate_ConfirmSerialNumber Function

Note: Strings are handled as zero-appended C strings.

Note: Because some types of serial number sets permit duplicate serial numbers, eSellerate_ConfirmSerialNumber does not check to see if a serial number is associated with more than one order.
