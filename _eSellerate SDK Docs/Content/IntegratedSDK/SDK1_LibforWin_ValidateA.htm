eSellerate_ValidateActivation

For eSellerate Product Activation purposes, validates that a serial number is activated and associated with the unique hardware identification of the computer in use. This is done by checking a hidden activation key on the computer.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

eSellerate_ErrorCode eSellerate_ValidateActivation ( publisherID, activationID, serialNumber )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString activationID – Product Activation Set ID string

eSellerate_InputString serialNumber – serial number string

Return Value

Description

eSellerate_NOENGINE

Returned if the eSellerateEngine cannot be found.

eSellerate_FAILURE

Returned if publisherID, activationID or serialNumber is NULL or if the eSellerateEngine encounters a fatal error during activation.

eSellerate_SUCCESS

Returned if the eSellerateEngine completes the action.

-25006

Returned if the eSellerateEngine completes the action, but activation fails because the activation key was unknown.

-25007

Returned if the eSellerateEngine completes the action, but activation fails because the activation key was not the one associated with serialNumber.

-25008

Returned if the eSellerateEngine completes the action, but activation fails because the activation key was invalid.
Return Values for eSellerate_ValidateActivation Function>

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.
