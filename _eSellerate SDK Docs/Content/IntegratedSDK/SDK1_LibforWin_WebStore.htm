eSellerate_WebStoreURL

Retrieves a web store URL stored by eSellerate SalesTrac Technology (eST) functions. See "eST downloads" in the eSellerate User's GuidePublishers Help index.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib
    Included in the development project.

Definition

eSellerate_ArraySize eSellerate_WebStoreURL ( publisherID, skuRefNum, resultBuffer, resultBufferSize )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString skuRefNum – SKU RefNum string

eSellerate_OutputString resultBuffer – character buffer to hold URL string

eSellerate_ArraySize resultBufferSize – size of character buffer

Return Value

Description

n

Length of the URL string put into resultBuffer. Value is zero if the URL is not found or if resultBufferSize is insufficient.

Return Values for eSellerate_WebStoreURL Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

skuRefNum

String containing the SKU RefNum, as found on the Sales Manager Available SKU Detail page. Required.

resultBuffer

Character buffer that eSellerate_WebStoreURL will fill in with the URL it retrieves. Required.

resultBufferSize

Size of character buffer provided as resultBuffer. Must be large enough to hold the URL retrieved. Required.

Parameters for eSellerate_WebStoreURL Function

Data Type

Definition

eSellerate_ArraySize

signed short

eSellerate_InputString

const char*

char*

Type Definitions for eSellerate_WebStoreURL Function

Note: Strings are handled as zero-appended C strings.

Example

#include "eSellerate.h"

char webStoreURL[256];

eSellerate_ArraySize webStoreURLSize = eSellerate_webStoreURL("PUB775527079", "SKU123456789", webStoreURL, 256);
