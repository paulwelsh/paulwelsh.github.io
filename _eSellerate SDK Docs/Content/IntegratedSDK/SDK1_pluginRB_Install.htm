InstallEngine

Copies the eSellerateEngine from the application to its proper location, if one of the following conditions are true:

  • No engine is present.
  • The engine that is present is older than the one carried by the application. In this case, the function replaces the older engine.
  • The engine that is present has been altered. In this case, the function replaces the altered engine.

This approach requires that the eSellerateEngine is built into the application as a resource. While this makes the application file bigger, it allows the application to not depend on an installer or other strategy to ensure that the engine is in place.

This function of the REALbasic plug-in option is available for Macintosh targets only.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • Resources
    Included in the development project. (If the project already contains a file named Resources, combine the resources of both files.)

    Because the eSellerate SDK does not include a file with this name, you will need to copy the file eSellerateEngine(Carbon).rsrc and rename the file Resources.

Definition

InstallEngine () as Integer

 

Return Value

Description

0

Returned if the eSellerateEngine is already properly installed, or if this function installed the engine.

-1

Returned if the eSellerateEngine is not already properly installed, and this function could not install the engine (perhaps because the eSellerateEngine was not built into the application as a resource).
Return Values for InstallEngine Function

Tip: For the best protection against engine tampering, you should compile your application with the latest eSelleratePlugin.rbx, and include the latest eSellerateEngine as a resource. Next, use the API call InstallEngine immediately before any call that requires the eSellerateEngine. Calling InstallEngine replaces an altered eSellerateEngine, if one is present.

Example

dim eSellerate as eSeller

dim resultCode as Integer

eSellerate = new eSeller

resultCode = eSellerate.InstallEngine()
