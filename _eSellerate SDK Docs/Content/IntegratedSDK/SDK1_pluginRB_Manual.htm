ManualActivateSerialNumber

For eSellerate Product Activation purposes, enables activation when the computer's configuration prevents the eSellerateEngine from accessing the eSellerate servers. Invokes a wizard and allows the user to manually activate the software using a web browser on the current computer or another computer with web access. Otherwise works like the ActivateSerialNumber function in that it confirms that a serial number originated from eSellerate and associates that serial number with the unique hardware identification of the computer in use.

Also see "Product Activation: manually activating serial numbers" in the eSellerate User's GuidePublishers Help index.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision
Windows Targets
  • eSellerateEngine.dllLocated in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

ManualActivateSerialNumber (

publisherID As String, – Publisher ID string

activationID As String, – Product Activation Set ID string

serialNumber As String – serial number string

) as Long

Return Value

Description

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if publisherID, activationID or serialNumber is "" or if the eSellerateEngine does not complete the action (such as when the user cancels the Manual Product Activation Wizard).

0

Returned if the eSellerateEngine completes the action.

Return Values for ManualActivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate User's GuidePublishers Help index.

serialNumber

String containing the serial number. Required.

Parameters for ManualActivateSerialNumber Function

Option Name

Description

"TitlePreface"

String containing a custom preface to the titles of all Manual Product Activation Wizard pages. Causes the titles to be "<Preface>: Product Activation." May be "", in which case the titles will simply be "Product Activation."

"OtherOptions"

String containing custom information to help users without web access activate their software. (This might be contact information for the publisher, who would visit http://activate.esellerate.net to obtain an Activation Key for the user.) The wizard will display up to seven lines for this string on Macintosh systems, and up to six lines on Windows. May be "", in which case the option to activate without web access will not be available to users.
Options for ManualActivateSerialNumber Function

Note: Options are set by SetOption. See SetOption.

Note: Unless there is a specific need for your application source code to directly invoke the Manual Product Activation Wizard, it is more efficient to use the ActivateSerialNumber function instead of ManualActivateSerialNumber.

When used with the proper askToConnect parameter, ActivateSerialNumber can handle normal (automatic) activations and also invoke the Manual Product Activation Wizard when needed. In contrast, ManualActivateSerialNumber can only handle manual activations, and consequently is limited to use in special circumstances.

Example

dim eSellerate as eSeller

dim serialNumber as String

dim resultCode as Integer

eSellerate = new eSeller

' ** TO DO: obtain serialNumber **

resultCode = eSellerate.ManualActivateSerialNumber("PUB775527079", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU", serialNumber)

' ** TO DO: handle error, or react to success **
