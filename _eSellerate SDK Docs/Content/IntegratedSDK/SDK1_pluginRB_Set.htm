SetOption

Assigns a value to options available to the following functions: Purchase, CheckForUpdate, SilentCheckForUpdate, Reinstall, ActivateSerialNumber, DeactivateSerialNumber and ConfirmSerialNumber.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision
Windows Targets
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

SetOption (

option as String, – defined name of option

setting as String – value to be assigned to option

)

Parameter

Description

option

String containing one of the defined option names listed in the next table. Required.

setting

String containing a value to be assigned to the option. Required.

Option Details for SetOption Function

Note: Option settings are "" (or "FALSE" when applicable) until set by SetOption.

Functions use the latest setting set by SetOption.

Call SetOption before calling the function(s) that will use it.

There is no need to reset an option setting between function calls, unless that setting needs to change.

Example

dim eSellerate as eSeller

dim resultCode as Integer

eSellerate = new eSeller

eSellerate.SetOption("ActivationID", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU")

resultCode = eSellerate.Purchase("PUB775527079", "ES123456789", "", "")
