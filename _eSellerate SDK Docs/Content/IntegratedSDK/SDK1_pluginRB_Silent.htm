SilentCheckForUpdate

Checks for an update using the eSellerateEngine without invoking the Software Delivery Wizard.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision
Windows Targets
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

SilentCheckForUpdate (

publisherID as String, – Publisher ID string

esellerID as String, – Integrated eSeller RefNum string

previewID as String – Preview Certificate string

) as Integer

Return Value

Description

-4

Returned if no update is available.

-1

Returned if publisherID or eSellerID is "" or if no active connection to the Internet is found and the value of the AskToConnect option is FALSE.

0

Returned if an update is available.

Return Values for SilentCheckForUpdate Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

esellerID

String containing the Integrated eSeller RefNum, as found on the Sales Manager Integrated eSeller Summary page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Integrated eSeller Summary page. Required to perform purchase in Preview mode. Pass "" in Active mode.

Parameters for SilentCheckForUpdate Function

Option Name

Description

"AskToConnect"

String that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass "TRUE" to enable display of the dialog; otherwise pass "FALSE".

"TrackingID"

String reserved for future functionality. Pass "".
Options for SilentCheckForUpdate Function

Note: Options are set by SetOption. See SetOption.

Caution: The SilentCheckForUpdate function cannot detect an active Internet connection on computers that are using an AirPort wireless network card. As a result, unless the function call prompts the user for permission to connect, the return value will be -1. You can work around this limitation by passing a TRUE value for the AskToConnect option, which will enable display of the connection dialog.

Example

dim eSellerate as eSeller

dim resultCode as Integer

eSellerate = new eSeller

resultCode = eSellerate.SilentCheckForUpdate("PUB775527079", "ES123456789", "")
