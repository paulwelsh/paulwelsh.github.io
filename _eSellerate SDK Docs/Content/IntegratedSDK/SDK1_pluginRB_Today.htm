Today

Returns the current date, expressed as the number of days since Jan. 1, 2000.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.

Definition

Today () as Integer

Return Value

Description

n

n is a date expressed as the number of days since Jan. 1, 2000.
Return Values for Today Function
