eWeb_StandardizeSerialNumber

Takes a Standard or Standard Enhanced serial number that has been validated with eWeb_ValidateSerialNumber and prepares the serial number for use with eWeb_ActivateSerialNumber.

This function bridges the gap between eWeb_ValidateSerialNumber, which tolerates certain characters if they are mistakenly entered in Standard or Standard Enhanced serial numbers, and eWeb_ActivateSerialNumber, which requires the exact serial numbers.

To prepare a validated serial number for activation, eWeb_StandardizeSerialNumber replaces certain characters that should never appear in a Standard or Standard Enhanced serial number (see the eWeb_ValidateSerialNumber notes), substitutes uppercase characters for lowercase ones and writes the result to a string.
