eWeb_DeactivateSerialNumber

For eSellerate Product Activation purposes, removes the association between a serial number and the unique hardware identification of the computer in use.

Required Files

  • EWSLib.h
    Included in the development project.
  • libEWS.a (standard) or
    libEWS-GC.a (garbage collecting)
    Included in the development project.
  • EWSMac.framework (standard) or
    EWSMac-GC.framework (garbage collecting)
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.

Definition

OSStatus eWeb_DeactivateSerialNumber ( publisherID, activationID, serialNumber )

publisherID – Publisher ID string

activationID – Product Activation Set ID string

serialNumber – serial number string


See eSellerate Return Values for details on return values.

Parameters for eWeb_DeactivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate Publishers Help index.

serialNumber

String containing the serial number. Required.

Note: All strings passed into the engine are UTF-8 encoded cStrings.

Example

#include "EWSLib.h"

char *serialNumber;

OSStatus resultCode;

/* TO DO: obtain serialNumber */
