eSellerate_Today

Returns the current date, expressed as the number of days since Jan. 1, 2000.

This function relies on the library libValidateUniversal.a, which you will need to link into your application.

Tip: eSellerate_Today is for finding the current date when you're using the validation library (libValidateUniversal.a) for Standard or Standard Enhanced serial numbers. To find the current date with the purchase library (libEWS.a/libEWS-GC.a), you can use eWeb_Today. See eWeb_Today.

Required Files

  • validateLib.h
    Included in the development project.
  • libValidateUniversal.a
    Linked into your application.

Definition

eSellerate_DaysSince2000 eSellerate_Today ()

Example

#include "validateLib.h"

eSellerate_DaysSince2000 timeStamp;

timeStamp = eSellerate_DaysSince2000 eSellerate_Today()
