eWeb_DisposePrefillData

Disposes of memory allocated by eWeb_NewPrefillData and used by any routines that consume prefill data. This function must be called on the prefill data for it to be properly disposed.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.
