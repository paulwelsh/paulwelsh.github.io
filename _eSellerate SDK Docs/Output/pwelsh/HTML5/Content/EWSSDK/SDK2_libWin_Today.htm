eWeb_Today

Returns the current date, expressed as the number of days since Jan. 1, 2000.

Tip: eWeb_Today is for finding the current date when you're using the purchase library (eWebClient.lib). To find the current date with the validation library for Standard or Standard Enhanced serial numbers (validateLibrary.lib), you can use eSellerate_Today. See eSellerate_Today.

Required Files

  • eWebLibrary.h
    Included in the development project.
  • eWebClient.lib
    Included in the development project.
  • (eWebClient.dll is not required for eWeb_Today, but it would be needed for other functions in the example code shown here.)
