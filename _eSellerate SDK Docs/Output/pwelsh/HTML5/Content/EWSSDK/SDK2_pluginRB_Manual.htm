ManualActivateSerialNumber

Launches the Embedded Web Store Engine to allow clients to manually activate a serial number. Typically this API is used when the current machine cannot get a network connection and a serial number needs to be activated.

Also see "Product Activation: manually activating serial numbers" in the eSellerate Publishers Help index.

Required Files

  • ewsPlugin.rbx
    Located in the REALbasic/Plugins folder.
Macintosh targets
  • EWSMac.framework
    Typically located in the /Library/Frameworks folder, but in some cases will be located in the ~/Library/Frameworks folder. It may be already placed in one or both of these locations by another eSellerate-enabled application.
Windows targets
  • eWebClient.dll
    Located in the eSellerate directory noted at Default Locations for eWebClient.dll. It may be already be placed here by another eSellerate-enabled application.

Definition

ManualActivateSerialNumber ( publisherID, activationID, serialNumber ) as Integer

publisherID as String – Publisher ID string

activationID as String – Product Activation Set ID string

serialNumber as String – serial number string


See eSellerate Return Values for details on return values.

Parameters for ManualActivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate Publishers Help index.

serialNumber

String containing the serial number. Required.
Options for ManualActivateSerialNumber Function

Option

Description

"TitlePreface"

String containing a custom preface to the titles of all Manual Product Activation Wizard pages. Causes the titles to be "<Preface>: Product Activation." May be "", in which case the titles will simply be "Product Activation."

"OtherOptions"

String containing custom information to help users without web access activate their software. (This might be contact information for the publisher, who would visit http://activate.esellerate.net to obtain an Activation Key for the user.) The wizard will display up to six lines for this string. May be "", in which case the option to activate without web access will not be available to users.

Note: Options are set by SetOption. See SetOption.

Note: Unless there is a specific need for your application source code to directly invoke the Manual Product Activation Wizard, it is more efficient to use the ActivateSerialNumber function instead of ManualActivateSerialNumber.

When used with the AskToConnect option set to TRUE, ActivateSerialNumber can handle normal (automatic) activations and also invoke the Manual Product Activation Wizard when needed. In contrast, ManualActivateSerialNumber can only handle manual activations, and consequently is limited to use in special circumstances.

Example

dim eSellerate as eStore

dim serialNumber as String

dim resultCode as Integer

eSellerate = new eStore

' ** TO DO: obtain serialNumber **

resultCode = eSellerate.ManualActivateSerialNumber("PUB775527079", "ACT123456789:9EUYLP-UZ8H-9J7J70-7FE4-6T18FY-72X9YK-FU9K-BDKLNQ-G53H-7FZTTU", serialNumber)

' ** TO DO: handle error, or react to success **
