GetSerialNumberCount

Organizes the serial number information obtained by a Wizard-invoking function and provides a count of the serial numbers.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory

Definition

GetSerialNumberCount As Long

Return Values for GetSerialNumberCount Function

Return Value

Description

-1

Returned if the Wizard-invoking function failed to complete.

0

Returned if the Wizard-invoking function completed, but no serial numbers were obtained, since none are needed.

n

Returned if the Wizard-invoking function completed, and n serial numbers were obtained.

Note: A Wizard-invoking function must be called before GetSerialNumberCount is called.

Call GetSerialNumberCount to organize data only once after calling a Wizard-invoking function.

Example

Dim eSellerate As eSeller

Dim serialNumberCount As Long

Set eSellerate = CreateObject("eSellerateControl.365")

' ** TO DO: call a Wizard-invoking function **

serialNumberCount = eSellerate.GetSerialNumberCount

' ** TO DO: handle error, or utilize results **

Set eSellerate = Nothing
