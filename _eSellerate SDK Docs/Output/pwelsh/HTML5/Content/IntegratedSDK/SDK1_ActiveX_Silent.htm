SilentCheckForUpdate

Checks for an update using the eSellerateEngine without invoking the Software Delivery Wizard.

Required Files

  • eSellerateControl365.dll
    Registered and located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place and register eSellerateControl365.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place and register eSellerateControl365.dll in the Windows directory
  • eSellerateEngine.dll
    Located in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

SilentCheckForUpdate (

publisherID As String, – Publisher ID string

esellerID As String, – Integrated eSeller RefNum string

previewID As String, – Preview Certificate string

askToConnect As Long, – ask to connect when no active connection is found

trackingID As String – reserved for future functionality

) As Long

Return Values for SilentCheckForUpdate Function

Return Value

Description

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if no active connection to the Internet is found, or if the value of the askToConnect parameter is true (non-zero) and no active connection could be made.

-4

Returned if no update is available.

0

Returned if an update is available.

Parameters for SilentCheckForUpdate Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

esellerID

String containing the Integrated eSeller RefNum, as found on the Sales Manager Integrated eSeller Summary page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Integrated eSeller Summary page. Required to purchase in Preview mode. Pass "" in Active mode.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a true (non-zero) value to enable display of the dialog; otherwise pass false (zero).

trackingID

String reserved for future functionality. Pass "".

Example

Dim eSellerate As eSeller

Dim resultCode As Long

Set eSellerate = CreateObject("eSellerateControl.365")

resultCode = eSellerate.SilentCheckForUpdate("PUB775527079", "ES123456789", "", 0, "")
