eSellerate_DeactivateSerialNumber

For eSellerate Product Activation purposes, removes the association between a serial number and the unique hardware identification of the computer in use.

Required Files

  • eSellerate.h
    Included in the development project.
  • eSellerateLibrary.lib(Carbon) or
    libeSellerate.a (Mach-O version)
    Included in the development project.
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision

Definition

eSellerate_ErrorCode eSellerate_DeactivateSerialNumber ( publisherID, activationID, serialNumber, askToConnect )

eSellerate_InputString publisherID – Publisher ID string

eSellerate_InputString activationID – Product Activation Set ID string

eSellerate_InputString serialNumber – serial number string

eSellerate_TrueOrFalse askToConnect – ask to connect when no active connection is found

Return Values for eSellerate_DeactivateSerialNumber Function

Return Value

Description

eSellerate_NOENGINE

Returned if the eSellerateEngine cannot be found.

eSellerate_FAILURE

Returned if publisherID, activationID or serialNumber is nil or if no active connection to the Internet is found and the value of the askToConnect parameter is false (zero).

eSellerate_SUCCESS

Returned if the eSellerateEngine completes the action.

-25001

Returned if deactivation fails because serialNumber was unknown.

-25003

Returned if deactivation fails because serialNumber is blacklisted.

-25004

Returned if deactivation fails because the product was returned.

-25006

Returned if deactivation fails because the activation key was unknown.

-25007

Returned if deactivation fails because the activation key was not the one associated with serialNumber.

-25008

Returned if deactivation fails because the activation key was invalid.

-25009

Returned if deactivation fails because serialNumber is already associated with another order.

-25010

Returned if the activation key is valid, but deactivation fails because the key was not found on the eSellerate servers.

-25011

Returned if deactivation fails because the deactivation limit was met.
Parameters for eSellerate_DeactivateSerialNumber Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

activationID

String containing an activation ID for use in eSellerate Product Activation. Required. See "Product Activation: setting up" in the eSellerate Publishers Help index.

serialNumber

String containing the serial number. Required.

askToConnect

Logical value that controls whether a dialog is displayed asking the user for permission to connect to the Internet when no active connection is found. Pass a true (non-zero) value to enable display of the dialog; otherwise pass false (zero).
