eSellerate_Today

Returns the current date, expressed as the number of days since Jan. 1, 2000. Available for use with either the purchase libraries or the validation libraries.

Required Files

  • esellerate.h (for use with eSellerateLibrary.lib(Carbon) or libeSellerate.a) or
    validate.h (for use with validateLibrary.lib(PPC) or libValidate.a)
    Included in the development project.
  • eSellerateLibrary.lib(Carbon)/libeSellerate.a (Mach-O version) or
    validateLibrary.lib(PPC)/libValidate.a (Mach-O version)
    Included in the development project.

Definition

eSellerate_DaysSince2000 eSellerate_Today ()

Return Values for eSellerate_Today Function

Return Value

Description

n

n is a date expressed as the number of days since Jan. 1, 2000.
