Reinstall

Invokes the Software Delivery Wizard to enable the user to reinstall software from a previous purchase.

Required Files

  • eSelleratePlugin.rbx
    Located in the REALbasic:Plugins folder.
Macintosh Targets
  • eSellerateEngine (Carbon)
    Located in the appropriate folder. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Mac OS 9.1 or later with CarbonLib 1.5 or later, place eSellerateEngine (Carbon) in System Folder:Application Support:MindVision
    • For Mac OS X, place eSellerateEngine (Carbon) in /Library/Application Support/MindVision
Windows Targets
  • eSellerateEngine.dl
    lLocated in the appropriate directory. It may be already placed here by another eSellerate-enabled application. The location depends on the platform:
    • For Windows Vista, place eSellerateEngine.dll in the ProgramData\eSellerate directory
    • For pre-Vista Windows versions, place eSellerateEngine.dll in the Windows directory

Definition

Reinstall (

publisherID as String, – Publisher ID string

esellerID as String, – Integrated eSeller RefNum string

previewID as String, – Preview Certificate string

errorURL as String URL – string shown in error messages

) as Integer

Return Values for Reinstall Function

Return Value

Description

-3

Returned if the user exits the eSellerateEngine before the action was completed.

-2

Returned if the eSellerateEngine cannot be found.

-1

Returned if the eSellerateEngine encounters a fatal error during reinstall.

0

Returned if the eSellerateEngine completes the action.

-25000

Returned if the eSellerateEngine completes the action, but activation fails because the ActivationID setting was unknown. (Only possible if the ActivationID option was set by SetOption.)

-25002

Returned if the eSellerateEngine completes the action, but no activation occurs because there is no serial number associated with the SKU. (Only possible if the ActivationID option was set by SetOption.)

-25003

Returned if the eSellerateEngine completes the action, but activation fails because the serial number is blacklisted. (Only possible if the ActivationID option was set by SetOption.)

-25004

Returned if the eSellerateEngine completes the action, but activation fails because the product was returned. (Only possible if the ActivationID option was set by SetOption.)

-25005

Returned if the eSellerateEngine completes the action, but activation fails because the activation limit was met. (Only possible if the ActivationID option was set by SetOption.)

-25009

Returned if the eSellerateEngine completes the action, but activation fails because the serial number obtained by calling Reinstall is already associated with another order. (Only possible if the ActivationID option was set by SetOption.)

-25012

Returned if eSellerate denies the activation request. (Only possible if the ActivationID option was set by SetOption.)
Parameters for Reinstall Function

Parameter

Description

publisherID

String containing the Publisher ID, as found on the Sales Manager Home page. Required.

esellerID

String containing the Integrated eSeller RefNum, as found on the Sales Manager Integrated eSeller Summary page. Required.

previewID

String containing the Preview Certificate, as found on the Sales Manager Integrated eSeller Summary page. Required to purchase in Preview mode. Pass "" in Active mode.

errorURL

String containing a contact URL for the Software Delivery Wizard to use in the text of certain message boxes when reporting an error. May be "".

Options for Reinstall Function

Option Name

Description

"OpeningTitle"

String containing a custom title of the opening page of the Software Delivery Wizard. May be "", in which case the system default title will be used.

"OpeningText"

String containing a custom first paragraph of the opening page of the Software Delivery Wizard. For Macintosh, the limit is four lines, or approximately 250 characters, depending on character width. For Windows, the limit is two lines, or approximately 150 characters, depending on character width. May be "", in which case the system default first paragraph will be used.

"ShowFolder"

String that controls whether the eSellerateEngine opens a window after the purchase to display downloaded files. Pass "TRUE" to open the folder; otherwise pass "FALSE".

"TrackingID"

String containing special information that marks the function call so that it can be tracked in reports. May be "". See "tracking IDs" in the eSellerate Publishers Help index.

"ActivationID"

String containing an activation ID for use in eSellerate Product Activation. May be "". Automatic activation occurs if the eSellerateEngine completes the action and returns a unique serial number for use with the ActivationID (barring any -25000 series errors). See "Product Activation: setting up" in the eSellerate Publishers Help index.

Note: Options are set by SetOption. See SetOption.

Example

dim eSellerate as eSeller

dim resultCode as Integer

eSellerate = new eSeller

resultCode = eSellerate.Reinstall("PUB775527079", "ES123456789", "", "")
